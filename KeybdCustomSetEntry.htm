KeybdCustomSetEntry

 

Sets a single custom key association entry in the custom key association list that applies to the keypad.

 

BOOL cdecl KeybdCustomSetEntry (WORD key, WORD output);

 

Parameters

WORD key

This parameter is a virtual key bit field that specifies the key to be customized.

WORD output

This parameter is a virtual key bit field that specifies the output key that replaces any occurrences of the virtual key specified by the “key” parameter.

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

For more information about CUSTOM_KEY see section Data Structures.
