DecodeGetFriendlyName

 

This gets the user friendly name associated with a decoding device.

 

BOOL cdecl DecodeGetFriendlyName(HANDLE hFile, DWORD * stringLen, LPWSTR friendlyName, DWORD maxLen );

 

Parameters:

HANDLE hFile

Handle to the decoding device to query. The CreateFile function returns this handle.

DWORD * stringLen

The length of friendlyName, not including the null terminator.  If NULL is passed into this parameter, then it is ignored.

LPWSTR friendlyName

Pointer to a buffer to receive the friendly name of the decoding device. The buffer must be large enough to store the entire string including the null terminator.

DWORD maxLen

Maximum length, in characters, of the string that may be copied into the buffer pointed to by string (not including the null terminator).

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
