Pharmacode Prefix

 

It enables the addition of the prefix character “A” to all Code 32 barcodes.  Pharmacode must be enabled for this parameter to function.

 

Syntax

Name:

IMAGER_PARAM_CODE39_CODE32PREFIX

Set by:

Value

Values:

IMAGER_PARAM_DISABLE,

IMAGER_PARAM_ENABLE

Default:

IMAGER_PARAM_DISABLE

 

 

Requirements

Header:

declared in IMAGER_PARAMETERS.H, include SE_PARAMETERS.H.
