WiFiGetActiveConfig

Gets the name of the first currently active/enabled profile

 

SDCERR WiFiGetActiveConfig(char *name, DWORD bufLen);

 

Parameters

char *name

Name of the desired profile

DWORD bufLen

number of characters allowed in the buffer

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on general failure

SDCERR_INVALID_PARAMETER if buffer is not large enough or name is NULL

 

Remarks

Supported devices: Skorpio X4, Falcon X4
