DecodeGoodRead

 

This produces the Good Read indication (audible and visual), as defined by the decoding configuration parameters.

 

BOOL cdecl DecodeGoodRead( DWORD beepType );

 

Parameters:

DWORD beepType

Indicates the type of Good Read beep to produce.  Valid values are listed in section Good Read Beep Types.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks:

If an invalid beep type is specified, FALSE is returned.
