getCodeID

 

public int getCodeID(int reqID)

 

This method reads the code ID of a data sample prior to calling a DecodeRead method. It allows the application to filter incoming samples by code ID.

 

Parameters:

reqID - The request ID to get the code ID of.

 

Returns:

Value indicating the code ID of the request. If there is an error, -1 is returned.
