DecodeGetDeviceCaps

 

This gets the capabilities of a specific decoding device.

 

DWORD cdecl DecodeGetDeviceCaps( HANDLE hFile );

 

Parameters:

HANDLE hFile

Handle to the decoding device to flush. The CreateFile function returns this handle.

 

Returns:

Zero on failure, otherwise the return value is a combination of bits which indicate the capabilities of the decoding device.

 

Remarks:

Bits are OR'ed together from the list in section Device Capabilities Mask.
