JoyaCradleReadConfigArea

It reads a config area from Joya Cradle Persistent Memory dedicated to cradle configuration data. First byte is the FastCharge enabling. Other bytes are to be specified for the future.

 

BOOL  cdecl  JoyaCradleReadConfigArea(JOYA_CRADLE_CONFIG* area);

 

Parameters

JOYA_CRADLE_CONFIG* area

area to read.

 

 

where:

typedef union

{

    BYTE fastChargeEnabled;

    BYTE data[32];

} JOYA_CRADLE_CONFIG;

 

 

 

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure. First byte gets if FastCharge is enabled or not.
