SoftSpotGetLockState

 

 

Indicates whether the SoftSpot position is locked.

 

BOOL cdecl SoftSpotGetLockState(BOOL* locked);

 

 

 

 

Parameters:

BOOL* locked

   Set to TRUE if SoftSpot will move with a finger drag, FALSE if SoftSpot is locked at its current position.

 

 

 

Returns:

TRUE if SoftSpot's move status was successfully retrieved, FALSE otherwise
