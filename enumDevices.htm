enumDevices

 

public static boolean enumDevices(int[] dev)

 

Returns the capabilities of all decoding devices.  

 

Returns:

An array of nine ints. This will return the capabilities of each decoding device in one of the ints, beginning with DCD1: in the first element, and ending with DCD9: in the last element. The values stored are combinations of bits, which indicate the capabilities of the decoding device. Bits are OR'd together from the Device Capabilities List. If there is an error retrieving the capabilities null is returned.
