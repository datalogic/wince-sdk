Datalogic .NET SDK

DecodeSetXMLParams(Method)

 

Syntax

 

public static bool DecodeSetXMLParams(String sBuffer, long nSize);

 

 

Class                           : Decode

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

This sets the current setting of programming parameters in an XML string

Parameters:

String sBuffer

String that will contain the XML string of configuration settings after this function call

long nSize

Number of bytes in the string.

 

Returns:

A value of true is returned on success, otherwise false is returned to indicate failure.

 

Remarks:

XML configuration schema is defined by Datalogic Configuration Utility.
