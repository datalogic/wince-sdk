DeviceGetBatteryThreshold

 

This function gets the value of the transitions between the main battery charge levels.  These settings affect when power notifications are generated, as well as what the returned battery status is when using Microsoft functions like GetSystemPowerStatusEx2 (see MSDN for details).

 

int cdecl DeviceGetBatteryThreshold( BAT_THRESHOLD *thresholds );

 

 

Parameters

BAT_THRESHOLD thresholds

This structure contains the values in which to get the low and critical thresholds.  For more information about BAT_THRESHOLD see section Data Structures.

 

Return Value

 A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
