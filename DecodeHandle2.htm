DecodeHandle

 

public DecodeHandle(int devCapabilities,

                                  int reserved)

                     throws DecodeException

 

Constructs a DecodeHandle object to the first decoding device with the specified capabilities.

 

Parameters:

devCapabilities - Specifies the input data type the decoder can read. (see Device Capabilities in APIEnum) This is an OR'd set of device capabilities.

reserved - This value is reserved. Use 0.

 

Throws:

DecodeException - if there is no device with the specified capabilities; or if the decoder handle cannot be created.
