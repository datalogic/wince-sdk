Datalogic .NET SDK

AddAction

 

Syntax

 

   public static UInt32 AddAction(SoftSpotAction newAction);

 

 

Class                           : SoftSpot

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Adds a new action to list of actions SoftSpot can cycle through and perform.

 

 

 

Parameters:

SoftSpotAction newAction         

    A  new SoftSpotAction to add.

    If actionType is ActionType.Scan, the application and arguments fields in newAction class are not used. For all action types, the actionId and order fields are not used

 

 

 

Returns:

ID of newly created action. If the add was not successful, a value of SOFTSPOT_ACTION_ID_INVALID is returned
