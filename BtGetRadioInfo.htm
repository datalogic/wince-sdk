BtGetRadioInfo

 

Gets bluetooth radio info, as returned in a BT_DEVICE structure.

 

BOOL cdecl BtGetRadioInfo( BT_DEVICE *radioInfo);

 

Parameters

BT_DEVICE *radioInfo

Pointer to a BT_DEVICE structure that has been allocated by the caller and will be filled in by the BtGetRadioInfo function.

 

Return Value

TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

Device must have bluetooth hardware.
