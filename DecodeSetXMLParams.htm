DecodeSetXMLParams

 

This sets the current setting of programming parameters from XML string

 

BOOL cdecl DecodeSetXMLParams (LPWSTR sBuffer, long nSize);

 

Parameters:

LPWSTR sBuffer

String pointer that will contain the XML string of configuration settings after this function call

long nSize

Number of bytes in the string pointer.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks:

XML configuration schema is defined by Datalogic Configuration Utility.
