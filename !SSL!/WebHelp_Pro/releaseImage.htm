releaseImage

 

public boolean releaseImage(int reqID)

 

This method releases the imager buffer associated with this DecodeHandle object and gives the driver control of the buffer. Once an application has been notified that image data is pending the driver will not write a new sample to the DecodeHandle object buffer until the application calls this method.

 

Parameters:

reqID - The request ID to release the image of.

 

Returns:

True is returned on success, otherwise false is returned.
