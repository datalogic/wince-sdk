DecodeCancelNotifyStart

 

Like posting requests, any posted notifications are automatically canceled when the device handle is closed. The notifications are posted to a specific decoding driver, so the notifications are limited to that device only. Notifications are sent when the device starts scanning.

 

BOOL DecodeCancelNotifyStart(HANDLE hFile);

 

Parameters:

HANDLE hFile

Handle to the decoding device from which a scanning start notification was requested.  The CreateFile function returns this handle. This must be the same handle used to request a previous notification.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
