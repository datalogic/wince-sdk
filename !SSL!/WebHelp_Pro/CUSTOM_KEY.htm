CUSTOM_KEY

 

It contains one custom key association.

 

Members:

 

WORD key

The default, un-customized keyboard code for a virtual key of the keyboard. This member specifies a default virtual key that is to be modified to a different virtual key when it occurs.

 

WORD output

 

The customized output keyboard code for a virtual key of the keyboard. The “key” value is replaced with the “output” value whenever the “key” value occurs.

 

Remarks:

The “key” and “output” members are bit fields that have the format seen in the table(s) below. The API does not support the Ctrl and Alt modifiers at the same time.  If both of these bits are set, the Ctrl state is used and the Alt state is ignored.
