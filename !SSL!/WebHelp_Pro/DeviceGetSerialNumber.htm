DeviceGetSerialNumber

 

This function retrieves the product serial number.

 

BOOL cdecl DeviceGetSerialNumber( WCHAR * serialNumber, int bufferLen );

 

 

Parameters

WCHAR * serialNumber

Pointer to a buffer that will receive the serial number. The buffer should be at least SERIAL_NUMBER_SIZE (see section Minimum Input Array Lengths) characters in length. The string returned will be null terminated.

int bufferLen

Number of characters allocated for serialNumber buffer.

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

If successful, then the buffer pointed to by serialNumber contains the serial number, if failed then serialNumber contains an empty string (the first element is the null termination character).
