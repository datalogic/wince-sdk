ISBT 128

 

It enables to decode the ISBT 128 barcode that is a variant of Code 128 used in the blood bank industry.

Syntax

Name:

IMAGER_PARAM_ISBT128

Set by:

Value

Values:

IMAGER_PARAM_DISABLE,

IMAGER_PARAM_ENABLE

Default:

IMAGER_PARAM_DISABLE

 

 

Requirements

Header:

declared in IMAGER_PARAMETERS.H, include SE_PARAMETERS.H.
