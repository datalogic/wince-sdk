Enable GS1 DataBar Omnidirectional

 

It enables the mobile computer to read the following GS1 DataBar symbologies:

Omnidirectional, Omnidirectional Stacked, Stacked, Truncated.

 

Syntax

Name:

BCD_PARAM_RSS_14

Set by:

Value

Values:

BCD_PARAM_DISABLE

BCD_PARAM_ENABLE

Default:

BCD_PARAM_DISABLE

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
