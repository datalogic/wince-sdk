DecodeReadString

 

This reads data that was used to fulfill a request previously posted to a decoding device.

BOOL cdecl DecodeReadString( HANDLE hFile, DWORD reqID, DWORD * stringLen, LPWSTR readString,  DWORD maxLen, CODE_ID * codeID );

Parameters:

HANDLE hFile

Handle to the decoding device to read data from. The CreateFile function returns this handle.

DWORD reqID

Request identifier of the request that has been satisfied with data to be read. The DecodePostRequest functions return this identifier.

DWORD * stringLen

The number of characters copied into the readString buffer, not including the null terminator, is returned to the DWORD pointed to by stringLen.  This value is meaningful only on success of the function. If NULL is passed into this parameter then it is ignored.

LPWSTR readString

Pointer to a buffer to receive the text data read by the decoding device. The buffer must be large enough to store the entire string including the null terminator.

DWORD maxLen

Maximum length, in characters, of the string that may be copied into the buffer pointed to by readString (not including the null terminator).

CODE_ID * codeID

Pointer to a variable which will receive the code identifier for the string. Null may be passed in as this parameter if the identifier is not of interest.

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.  Specifying a request ID that has not yet been fulfilled will cause the function to return FALSE.

 

Remarks:

If the buffer is not large enough to contain the entire text data associated with the request ID, then the number of characters present in the text data is returned (not including a null terminator) into the DWORD pointed to by stringLen.

The text data returned will include symbology identifier characters, if enabled, appended to the beginning of the decoded data. It does not include any label prefix or label suffix characters. It is valid to have this function return zero length string (stringLen), since null strings may be read by a decoding device.
