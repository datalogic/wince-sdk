IOCTL_HAL_DISABLE_WAKE

This IOCTL disables an interrupt source from waking the system.

 

 

It has to be used with the KernelIoControl function:

 

        BOOL KernelIoControl(

          DWORD dwIoControlCode,

          LPVOID lpInBuf,

          DWORD nInBufSize,

          LPVOID lpOutBuf,

          DWORD nOutBufSize,

          LPDWORD lpBytesReturned

        );

 

 

Parameters

dwIoControlCode

[in] Set to IOCTL_HAL_DISABLE_WAKE.

 

lpInBuf

[in] Set to the required SYSINTR, which is the source of the interrupt.

 

nInBufSize

[in] sizeof(DWORD)

 

lpOutBuf

Ignored.

 

nOutBufSize

Ignored.

 

lpBytesReturned

Ignored.

 

 

Return Value

TRUE -  indicates success

FALSE - indicates failure

 

 

Requirements

Header:

pkfuncs.h, nkintr.h

 

 

Example

#include <windows.h>

#include <pkfuncs.h>

#include <nkintr.h>

....

DWORD dwSysIntr = SYSINTR_RTC_ALARM;

KernelIoControl(IOCTL_HAL_DISABLE_WAKE, &dwSysIntr,sizeof(dwSysIntr), NULL, 0, NULL);
