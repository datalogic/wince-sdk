setWedge

 

public static boolean setWedge(int inputType,

                                               boolean enable)

 

Enables or disables the keyboard wedge feature for a particular type of input data.  

 

Parameters:

inputType - Indicates the type of input data whose wedge setting is to be altered. (one of the Input Data Types in APIEnum)

enable - Specifies whether or not the wedge feature is enabled. If false, the feature is disabled, otherwise it is enabled.

 

Returns:

True is returned on success, otherwise false is returned. If an invalid input type is specified, false is returned.
