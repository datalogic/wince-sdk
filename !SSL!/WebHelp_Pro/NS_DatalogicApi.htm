Datalogic.API (Namespace)

 

Assembly                   : Datalogic.API.DLL

 

 

Description

This namespace expose all methods, static functions, enumerations and costants useful to manage the Datalogic’s decoding device using the .NET framework. In the table below you can find the complete list of the classes exported.

DecodeException
