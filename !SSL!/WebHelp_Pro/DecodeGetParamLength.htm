DecodeGetParamLength

 

DWORD cdecl DecodeGetParamLength(DWORD index);

 

Parameters:

DWORD index

The index of the label programming parameter to get the length of.

 

Returns:

On success, the string length (number of WCHARs) of the parameter's value is returned.
On error, including a parameter index that does not support string values, -1 is returned.

 

Remarks:

This method is meant to be used with index arguments that reference parameters that are a string of Unicode characters as indicated in General Decoding Definitions.  It can be used to find the size of the WCHAR buffer needed for a call to DecodeGetParamString.  The WCHAR buffer should be at least the parameter length plus one for the null string termination.

This method is for use with the following index arguments:

LABEL_PREFIX
LABEL_SUFFIX
LABEL_SEPARATOR.
