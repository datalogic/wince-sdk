Supplemental Redundancy

 

With Autodiscriminate UPC/EAN Supplementals selected, this option adjusts the number of times a symbol without supplementals is decoded before notifying to the application. The range is from 2 to 20 times. Five or above is recommended when decoding a mix of UPC/EAN symbols with and without supplementals, and the autodiscriminate option is selected.

 

 

Syntax

Name:

BBCR_PARAM_DECODE_UPC_EAN_SUPPLEMENTAL_REDUNDANCY

Set by:

Value

Values:

from 2 to 20

Default:

7 (Serial One Dimensional Barcode Engine) 20 (Serial Barcode Raster Engine)

 

 

Requirements

Header:

declared in BBCR_PARAMETERS.H, include SE_PARAMETERS.H.
