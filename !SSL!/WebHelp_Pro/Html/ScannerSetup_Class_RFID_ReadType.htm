Read Type

 

This parameter allows selecting the type of reading.

What the reading buffer returns by the methods of DLScanner class i.e. DLScanner::getLabelText or sLabelText property of the DatalogicScanner ActiveX) will contain:

 

  • only the identifier;

  • the identifier + the data buffer;

  • the data buffer + the identifier;

  • only the data buffer

 

 

Syntax

Name:

RF_PARAM_READ_TYPE

Set by:

Value

Values:

RF_PARAM_ID

RF_PARAM_ID_AND_DATA_BUFFER

RF_PARAM_DATA_BUFFER_AND_ID

RF_PARAM_DATA_BUFFER

Default:

RF_PARAM_ID

 

 

Requirements

Header:

declared in RF_Parameters.h, include SE_PARAMETERS.H.
