Symbology Identifier

 

It adds a symbology identifier of one or more characters at the beginning or at the end of the barcode string returned by the scanner. In this way, it is possible to know the code symbology the decode code belongs to. The identifier can be managed as follows:

  • No identifier;

  • Datalogic identifier at string beginning;

  • Datalogic identifier at string end;

  • AIM identifier at string beginning;

  • AIM identifier at string end.

 

 

NOTE

Note that the Datalogic identifier is the same returned by the DLScanner::getIdentifier method.

 

 

Syntax

Name:

BCD_PARAM_BARCODE_SYMBOLOGY_IDENTIFIER

Set by:

Value

Values:

BCD_PARAM_NO_IDENTIFIER

BCD_PARAM_DATALOGIC_IDENTIFIER_AT_BEGINNING

BCD_PARAM_DATALOGIC_IDENTIFIER_AT_END

BCD_PARAM_AIM_IDENTIFIER_AT_BEGINNING

BCD_PARAM_AIM_IDENTIFIER_AT_END

Default:

BCD_PARAM_NO_IDENTIFIER

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
