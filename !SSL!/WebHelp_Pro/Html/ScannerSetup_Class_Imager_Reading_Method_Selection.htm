Reading Method Selection

 

It allows managing two different barcode reading methods:

 

Method1

 

This method works well in most standard applications. It is the default setting.

 

 

Method2

 

This method can be useful when reading small codes positioned on an extensive background.

 

 

Syntax

Name:

IMAGER_PARAM_READING_METHOD

Set by:

Value

Values:

IMAGER_PARAM_READING_METHOD1

IMAGER_PARAM_READING_METHOD2

Default:

IMAGER_PARAM_READING_METHOD1
