GlobalSettings

GlobalSettings struct

 

Members:

 

DWORD structSize

Must be set to the sizeof( ) the struct.

int roamRssiThreshold

Roaming RSSI Threshold, in dBm. Value must be between ROAM_RSSI_THRESH_MIN and ROAM_RSSI_THRESH_MAX. Also, value cannot be greater than bgScanLevel value.

DWORD roamRetryThreshold

Roaming Retry Threshold, number of retries. Value must be between ROAM_RETRY_THRESH_MIN and ROAM_RETRY_THRESH_MAX.

DWORD roamMinConnTime

Roaming Minimum Connection Time, in seconds. Value must be between ROAM_MIN_CONN_TIME_MIN and ROAM_MIN_CONN_TIME_MAX.

int roamRssiDiff

Roaming RSSI Difference, in dBm. Value must be between ROAM_RSSI_DIFF_MIN and ROAM_RSSI_DIFF_MAX.

int bgScanLevel

Background Scan Level, in dBm. Value must be between BGSCAN_LEVEL_MIN and BGSCAN_LEVEL_MAX.

DWORD bgScanInterval

Background Scan Interval, in seconds. Value must be between BGSCAN_INTERVAL_MIN and BGSCAN_INTERVAL_MAX.

BYTE channelList[MAX_NUM_OF_CHANNELS]

List of enabled channels. Max size is MAX_NUM_OF_CHANNELS.

DWORD channelCount

The number of channels in the channelList array that are valid.

BOOL ccxEnable

CCX Enabled boolean

BOOL cckmEnable

CCKM Enabled boolean

Edot11Mode phyMode

Current Wi-Fi PHY (physical layer) mode

BOOL powerSaveEnable

Power Save Enabled boolean
