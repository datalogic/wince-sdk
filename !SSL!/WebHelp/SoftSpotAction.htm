SoftSpotAction

The structure holds information about a given SoftSpot action that can be performed by SoftSpot.

 

 

Members:

 

 

DWORD actionId; Unique ID of given action

WCHAR title[SOFTSPOT_MAX_TITLE_LENGTH];

 Human-readable action title
WCHAR icon[MAX_PATH];   

Full path to icon image used for Action. 32-bit (RGBA) PNG files are the only types of image files supported

SOFTSPOT_ACTION_TYPE actionType;

Type of action (scan or launch app)

WCHAR application[MAX_PATH];

Full path to executable to launch

WCHAR arguments[MAX_PATH];

Arguments to pass to executable on launch
DWORD order;           Order action is displayed in Action list
