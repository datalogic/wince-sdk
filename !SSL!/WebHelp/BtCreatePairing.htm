BtCreatePairing

 

Create a pairing based on a string BT UUID of a device.

 

BOOL cdecl BtCreatePairing( WCHAR *addrStr, int profile, int portIndex, UCHAR *pin);

 

Parameters

WCHAR *addrStr

Null terminated string containing the MAC address of the radio to pair to.

int profile

A value representing the profile to connect to (see 5.4.1 Bluetooth profile values).

int portIndex

index of virtual serial port to create. Only used with BT_PROFILE_PRINTER and BT_PROFILE_SERIAL.

UCHAR *pin

Null terminated array of ASCII characters containing the pin. If NULL no authentication is used. The pin must be between BT_MIN_PIN_SIZE and BT_MAX_PIN_SIZE.

 

Return Value

TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

On failure an additional error message can be found by calling GetLastError().

 

ERROR_BUSY

Driver is busy performing an inquiry.

ERROR_NO_MORE_DEVICES

Attempt to add more than MAX_DEVICES pairings.

ERROR_INVALID_PARAMETER

The target device does not support serial or does not exist.

ERROR_INVALID_PASSWORD

The PIN is invalid.

 

Device must have bluetooth hardware. For Joya, only serial pairings are allowed and only one pairing is allowed.
