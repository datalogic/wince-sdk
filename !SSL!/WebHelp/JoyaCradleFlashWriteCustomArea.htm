JoyaCradleWriteCustomArea

It writes a custom byte array to Joya Cradle Persistent Memory dedicated to custom data.

 

BOOL  cdecl  JoyaCradleWriteCustomArea(const BYTE* area, unsigned long size);

 

Parameters

BYTE* area

Array of byte to write.

 

unsigned long size

Size of the array (max valkue is 512).

 

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
