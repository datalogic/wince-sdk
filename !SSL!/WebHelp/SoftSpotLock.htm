SoftSpotLock

 

 

Locks or unlocks the SoftSpot position. In the locked state, the user cannot move SoftSpot.

 

BOOL cdecl SoftSpotLock(BOOL move);

 

 

 

 

Parameters:

BOOL move       

    Set to TRUE if SoftSpot should be allowed to move, FALSE to keep SoftSpot locked at it's current position

                                              

 

Returns:

TRUE if SoftSpot's move status was successfully set, FALSE otherwise
