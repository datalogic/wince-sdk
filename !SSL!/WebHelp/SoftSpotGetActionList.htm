SoftSpotGetActionList

 

 

Get an ordered list of actions in SoftSpot's action list

 

BOOL cdecl SoftSpotGetActionList(SoftSpotAction *actionIds, int *count);

 

 

 

 

Parameters:

SoftSpotAction *actionIds

    Pointer to array of SoftSpotAction structs to be populated.

 

int *count

    Contains the maximum number of actions that should be retrieved. This value will be updated by the function to indicate how many actions were actually loaded into the array

 

 

 

Returns:

TRUE if the list of actions could be successfully retrieved. FALSE otherwise
