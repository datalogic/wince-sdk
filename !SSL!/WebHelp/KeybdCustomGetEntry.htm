KeybdCustomGetEntry

 

This function gets a single custom key association from the custom key association list that applies to the keypad. This function is used to find out if a particular virtual key has been customized.

 

BOOL cdecl KeybdCustomGetEntry (WORD key, WORD* output);

 

Parameters

WORD key

This parameter is a virtual key code bit field as defined by the CUSTOM_KEY data type.
This value specifies which custom key code is returned.

WORD output

This parameter points to a WORD that receives the custom key output for the virtual key specified by the “key” parameter.

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

If the value pointed to by the “output” parameter is set to the “key” parameter value, it indicates that the key specified by the “key” parameter has not been customized.

For more information about CUSTOM_KEY see section Data Structures.
