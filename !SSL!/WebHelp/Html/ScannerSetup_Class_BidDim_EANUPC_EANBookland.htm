Enable EAN Bookland

 

It enables the mobile computer to read the EAN Bookland code symbology.

 

 

Syntax

Name:

BBCR_PARAM_BOOKLAND_EAN

Set by:

Value

Values:

BBCR_PARAM_DISABLE

BBCR_PARAM_ENABLE

Default:

BBCR_PARAM_DISABLE

 

 

Requirements

Header:

declared in BBCR_PARAMETERS.H, include SE_PARAMETERS.H.
