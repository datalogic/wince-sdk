Data Address

 

This parameter specifies the address (bytes offset or block number, see RF_PARAM_DATA_BUFFER_ADDRESSING_MODE) on the RFID memory from which the RFID driver reads the number of bytes specified in the parameter RF_PARAM_DATA_BUFFER_DIMENSION.

 

 

Syntax

Name:

RF_PARAM_DATA_BUFFER_ADDRESS

Set by:

Value

Values:

unsigned long: from 0 to 65535

Default:

0

 

The highest address depends on the RFID label features: it corresponds to the number of blocks multiplied by the number of bytes per block minus 1.

 

 

Requirements

Header:

declared in RF_Parameters.h, include SE_PARAMETERS.H.
