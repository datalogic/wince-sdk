QRCodeEncoding (Enumeration)

 

Syntax

 

   public enum QRCodeEncoding: int

   {

        Format_8859     = 0,

        Format_UTF8     = 1,

        Format_ShiftJIS = 2,

   }

 

 

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Description

Options for the QR Code Encoding Parameter (QRCODE_ENCODING).

 

 

Name

Value

Description
Format_8859

0

Data is encoded in 8859-1/ASCII format.

Format_UTF8

1

Data is encoded in UTF-8 format.

Format_ShiftJIS

2

 Data is encoded in Shift-JIS format.
