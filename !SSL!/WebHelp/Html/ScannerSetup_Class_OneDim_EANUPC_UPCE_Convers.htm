UPC E Conversion Type

 

It defines the conversion type to be applied to the UPC E codes. Possible selections are:

  • no conversion;

  • conversion to EAN 13;

  • conversion to UPC A.

 

 

Syntax

Name:

BCD_PARAM_UPC_E_CONVERSION_TYPE

Set by:

Value

Values:

BCD_PARAM_UPC_E_NO_CONVERSION

BCD_PARAM_UPC_E_TO_EAN_13

BCD_PARAM_UPC_E_TO_UPC_A

Default:

BCD_PARAM_UPC_E_TO_EAN_13

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
