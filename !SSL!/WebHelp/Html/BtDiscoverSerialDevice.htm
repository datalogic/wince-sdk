BtDiscoverSerialDevices

 

This function returns information about all discoverable devices in range that support the serial (SPP) profile.  This function can be used with BtCreateSerialPairing() to form a connection to a serial BT device, such a printer.

 

BOOL cdecl BtDiscoverySerialDevices( BT_DISCOVERY_INFO *info);

 

 

Parameters

BT_DISCOVERY_INFO *info

Structure containing information about discovered devices

 

 

Return Value

TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

This function will not return until the device discovery process is complete. On failure an additional error message can be found by calling GetLastError().

 

ERROR_INVALID_HANDLE

No BT hardware, discovery already in progress or input parameters not valid.

 

Device must have Bluetooth hardware.
