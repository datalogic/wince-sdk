AccelerometerStartMsg

Request Windows Messages from Accelerometer GSensor.

 

BOOL AccelerometerStartMsg(HANDLE hAccDevice, DWORD dwPollingInterval, HWND hWnd, UINT uMsg);

 

 

Parameters

HANDLE hAccDevice

Handle to the Accelerometer GSensor returned by AccelerometerOpen()

DWORD dwPollingInterval

The accelerometer polling interval in units of milliseconds. This value will be rounded to the nearest 10ms increment by the driver

HWND hWnd

Handle to a window whose window procedure is to receive a message when the request is fulfilled

UINT uMsg

Specifies the message to be posted when the request is fulfilled. This should be a value of WM_USER (defined in winuser.h) or higher

 

Return Value

On success, TRUE is returned. On error, FALSE is returned.

 

Remarks

An application should either use AccelerometerStartMsg or AccelerometerStartEvent.

When posted to the specified window, the lParam and wParam value of the message are set as follows:

 

Value

Description
lParam Handle to the Accelerometer GSensor (hAccDevice)
wParam
