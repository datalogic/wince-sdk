WiFiGetCurrentStatus

Retrieve the current status information

 

SDCERR WiFiGetCurrentStatus(CF10G_STATUS *status);

 

Parameters

CF10G_STATUS *status

Pointer pre-allocated conifg status struct to be filled

 

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on failure

 

Remarks

Supported devices: Skorpio X4, Falcon X4
