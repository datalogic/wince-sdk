BtInitialize

 

Initializes the Bluetooth Driver.

 

BOOL cdecl BtInitialize( void );

 

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

On failure an additional error message can be found by calling GetLastError().

 

 

Return Values

Description

ERROR_ALREADY_EXISTS

Driver already initialized

INVALID_HANDLE_VALUE

Failure to load driver, or lack of Bluetooth hardware
