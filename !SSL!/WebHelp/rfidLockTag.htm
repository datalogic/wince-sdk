rfidLockTag

 

public int rfidLockTag(int tagType,

                               int password,

                               int timeout)

 

This locks the information in a tag's memory from being changed. It applies to any tag within the RF field of the reader. It is up to the application and/or user to make sure only the desired tags are affected. The method is depricated. Use rfidLockTagEx.

 

Parameters:

tagType - The type of tag to lock.

password - Password to be used to for locking the tag(s). This must match the password in the tag(s) in order for the tag(s) to be locked.

timeout - Timeout in milliseconds to attempt to lock the specified tag.

 

Returns:

An integer indicating the success or failure of the method. See the RFID error codes in APIEnum.
