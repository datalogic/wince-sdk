DecodeSetParams

 

This sets the values of one or more label programming parameters.  

 

BOOL cdecl DecodeSetParams( DCD_PARAM * paramArray, int paramCount );

 

Parameters:

DCD_PARAM * paramArray

Pointer to an array of DCD_PARAM structures (see section Data Structures) indicating the parameters to set. Each array element contains the index of a parameter to set, and its new value.

int paramCount

Number of elements in the array pointed to by paramArray.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks:

If any unsupported index is used, or if an attempt is made to set a parameter to a value outside its valid limits, none of the parameter changes will take effect, and FALSE is returned. When the function fails, GetLastError will return one of the error codes listed in Section Error Codes.

Parameters which do not have values (such as DEFAULTS_FACTORY) may be included in the input array.  In this case, the value member is ignored, and the value of all parameters are updated immediately based on the parameter chosen.  Additional parameters which follow in the array will be processed after the global change takes effect.  

For example, calling DecodeSetParams with the following input:

{CODE39_MIN_LENGTH, 5}

{DEFAULT_FACTORY, 0}

{CODE39_MAX_LENGTH, 10}

will result in all parameters set to factory defaults, with Code 39 minimum length set to 1 (the factory default) and Code 39 maximum length set to 10.

An empty array assigned to the argument ‘paramArray’ is an invalid input.
