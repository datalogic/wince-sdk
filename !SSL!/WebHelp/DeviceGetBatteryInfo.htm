DeviceGetBatteryInfo

 

This function retrieves information about the main battery that is not returned using Microsoft built-in functionality.  See section BATINFO, P_BATINFO for the BATINFO structure definition.

 

BOOL cdecl DeviceGetBatteryInfo( BATINFO * info );

 

 

Parameters

BATINFO * info

Fills the structure pointed to by info with various information about the main battery (see section BATINFO, P_BATINFO).

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
