AccelerometerRead

This reads data that was used to fulfill a request previously posted to the Accelerometer GSensor.

 

BOOL cdecl AccelerometerRead(HANDLE hAccDevice, *PGACC_DATA resultValue);

 

 

Parameters

HANDLE hAccDevice

Handle to the Accelerometer GSensor returned by AccelerometerOpen()

*PGACC_DATA resultValue

Returned value of x.y.z data from Accelerometer (See Section 5.2.6 for definition of GACC_DATA)

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

It is the responsibility of the calling function to allocate the resultValue.
