BtTerminate

 

De-initializes the Bluetooth driver.

 

BOOL cdecl BtTerminate(void);

 

Return Value

A value of TRUE is returned on success. FALSE is returned if the file HANDLE is not open.

 

Remarks

On failure an additional error message can be found by calling GetLastError()

 

 

Return Values

Description

INVALID_HANDLE_VALUE

Driver not previously loaded

 

 

Device must have bluetooth hardware.
