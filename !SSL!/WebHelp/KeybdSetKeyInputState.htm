KeybdSetKeyInputState

 

This function sets the key input state for the keyboard.

 

BOOL cdecl KeybdSetKeyInputState( DWORD state );

 

 

Parameters

DWORD state

A combination of OR'd values that indicate the state of the keyboard to set. Possible values are listed in section Keyboard input state bit definitions.

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

State key down flags are ignored and the function returns TRUE.
