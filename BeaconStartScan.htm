BeaconStartScan

 

This will cause beacon scanning to start, if it has not been already started.

 

HANDLE cdecl BeaconStartScan(HANDLE beaconEvent);

 

Parameters

HANDLE beaconEvent

Handle to the event that is signaled when a beacon is scanned.  This event should not be closed until after BeaconStopScan is called.

Return Value

On success, returns a handled that can be used to other beacon functions, else this function returns INVALID_HANDLE_VALUE.

Remarks

Supported Devices: Joya Touch
