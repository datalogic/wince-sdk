setTriggerType

 

public static boolean setTriggerType(int trigger,

                                                     int inputType)

 

Sets the type of input data acquired when a trigger is activated.  

 

Parameters:

trigger - Specifies the trigger to set. (one of the Trigger Identifiers in APIEnum)

inputType - Specifies the input data type to begin acquiring when the trigger is activated. (one of the Input Data Types in APIEnum) Zero may also be used to disable the trigger.

 

Returns:

True is returned on success, otherwise false is returned. Specifying an invalid trigger or input type will return false.
