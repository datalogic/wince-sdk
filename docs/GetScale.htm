Datalogic .NET SDK

GetScale

 

Syntax

 

   public static float GetScale();

 

 

Class                           : SoftSpot

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Get the current size (scale) of the SoftSpot.

 

Returns:         

    A float that will be filled in with a value between SOFTSPOT_SCALE_MIN and SOFTSPOT_SCALE_MAX that indicates the current scaling factor of the SoftSpot.

    For example, 1.2 would be 20% larger than the default size (which is 1.0).
