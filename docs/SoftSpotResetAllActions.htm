SoftSpotResetAllActions

 

 

Resets the list of actions SoftSpot can perform, according to value of resetState parameter.

 

BOOL cdecl SoftSpotResetAllActions(SOFTSPOT_ACTION_RESET_STATE resetState);

 

 

 

 

Parameters:

SOFTSPOT_ACTION_RESET_STATE resetState     

    State to reset the list to.

                                              

 

 

Returns:

TRUE if reset was performed, FALSE otherwise
