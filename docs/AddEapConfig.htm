DEPRECATED

This function is present in the SDK, but the WiFiAddConfig() should be used with a SDCConfig struct instead to setup EAP profiles.

 

WiFiAddEapConfig

Edits the current EAP profile

 

SDCERR WiFiAddEapConfig(EapConfig *cfg);

 

Parameters

EAPConfig *cfg

The EAP configuration struct to set as the current profile

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on failure

 

 

 

Example: Adding an EAP-PEAP network

 

  EapConfig cfg;

  WiFiInitEapConfig(&cfg);

 

  strcpy(cfg.SSID, "myEapNetwork");

  cfg.CipherAlg = DOT11_CIPHER_ALGO_CCMP; // AES

  cfg.AuthAlg = DOT11_AUTH_ALGO_RSNA; // WPA2

  cfg.EapType = PPP_EAP_PEAP;

  cfg.CredType = PASSWORD;

  cfg.CredSource = FROM_PROFILE;

  cfg.InnerMethod = EAP_MsChapV2;

  strcpy(cfg.UserName, "myUser");

  strcpy(cfg.Password, "myPass");

 

  SDCERR err = WiFiAddEapConfig(&cfg);

  printf("result=%d\n", err);
