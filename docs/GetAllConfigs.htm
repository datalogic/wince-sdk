WiFiGetAllConfigs

Retrieves up to MAX_CFGS existing Wi-Fi profiles from the device. WEP/PSK keys or EAP authenication data are not returned.

 

SDCERR WiFiGetAllConfigs(SDCConfig *cfgs, DWORD *num);

 

Parameters

SDCConfig *cfgs

Pointer to array of MAX_CFGS configurations

 

DWORD *num

Return value containing the number of configs pointed to by cfgs

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on failure

 

Remarks

Supported devices: Skorpio X4, Falcon X4
