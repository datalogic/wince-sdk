Example Path:  <Datalogic SDK installation folder>\DotNet\Samples\

 

 

Description

This table describe the examples you can find in the “Samples” path giving an outline of what they do and on what devices they can be executed.

This table summarize the examples you can find in the “Samples” path.

There are four solution, one for each class of the Datalogic.API namespace and inside the solution there are one or more projects that you can

compile in stand alone mode or together because are independent each other.

 

Compiling requirements:

  • Visual Studio 2005 (Visual C#);

  • Debug Mode compiling;

  • Target Device might be: "Intel PXA27x Dev Platform: ARMV4I_Release" or "Windows CE 6 SDK ARMV4I Device" according to the example;

  • Assembly: Datalogic.API.dll v1.12.0.0.
