Enable GS1 DataBar Expanded

 

It enables the mobile computer to read the GS1 DataBar Expanded symbology.

 

Syntax

Name:

BBCR_PARAM_RSS_EXPANDED

Set by:

Value

Values:

BBCR_PARAM_DISABLE

BBCR_PARAM_ENABLE

Default:

BBCR_PARAM_DISABLE

 

 

Requirements

Header:

declared in BBCR_PARAMETERS.H, include SE_PARAMETERS.H.
