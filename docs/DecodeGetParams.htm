DecodeGetParams

 

This gets the values of one or more label programming parameters.

 

BOOL cdecl DecodeGetParams( DCD_PARAM * paramArray, int paramCount );

 

Parameters:

DCD_PARAM * paramArray

Pointer to an array of DCD_PARAM structures (see section Data Structures) indicating the parameters to get. Each array element's index member is initialized with the index of a parameter to get, and the value member is filled in upon return with the value of the parameter.

int paramCount

Number of elements in the array pointed to by paramArray.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks:

If an unsupported index is included in the parameter array, its value will be set to -1, but the return value will still be TRUE.
