StopScanListener (Method)

 

Syntax

 

public bool StopScanListener();

 

 

Class                           : DecodeEvent

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Description

The StopScanListener method will make DecodeEvent stop listening to the decoder.

 

 

Parameters Table

 

Type

Parameter

Description

[in]

void

 

-

 

 

 

 

[return]

bool

 

If DecodeEvent is not listening to the decoder, false will be returned, otherwise true will be returned.

[on error]

-

 

-

 

 

Example

 

   private void MainDlg_Closing(object sender, System.ComponentModel.CancelEventArgs e)

   {

       // If our instance of DcdEvent is listening to the decoder, we need to make

       // sure we tell DcdEvent to stop listening.

       if(dcdEvent.IsListening)

       {

             dcdEvent.StopScanListener();

       }

   }

 

 

 

Remarks

Make sure to call Datalogic.API.DecodeEvent.StopScanListener to stop listening to the decoder.
