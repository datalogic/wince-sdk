DEPRECATED

This function is present in the SDK, but the SDCConfig struct should be used instead to setup EAP profiles. There is no initialization function for the SDCConfig struct.

 

WiFiInitEapConfig

Initializes fields in struct to default values. Required.

 

SDCERR WiFiInitEapConfig(EapConfig *cfg);

 

Parameters

EAPConfig *cfg

The EAP configuration struct to initialize

 

cfg is initialized with the following values:

 

SSID[0] = 0;

Broadcast = 1;

Protect = 0;

ImportPACFilePath[0] = 0;

pacAIDinfo[0] = 0;

Cckm = 0;

CipherAlg = DOT11_CIPHER_ALGO_IHV_END;

SharedKey[0] = 0;

AuthAlg = DOT11_AUTH_ALGO_IHV_END;

EapType = PPP_EAP_MSCHAPv2;

PmkCache = 0;

CredType = CT_NOT_USED;

CredSource = CS_NOT_USED;

ValidateServer = 0;

ServerCertName[0] = 0;

ClientCertName[0] = 0;

InnerMethod = INNER_NOT_USED;

AutoPAC = 1;

EncryptID = 1;

UserName[0] = 0;

Password[0] = 0;

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on failure

 

Remarks

Supported devices: Skorpio X4, Falcon X4
