CvtBtStringToAddr

 

Convert Bluetooth String to an address

 

BOOL cdecl CvtBtStringToAddr( WCHAR *, BT_ADDR *, DWORD len );

 

 

Parameters

WCHAR*

Pointer to string to be converted

 

BT_ADDR*

Pointer to bluetooth address to hold result of converting the string to a BT_ADDR.

 

DWORD len

Length of the string to be converted

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
