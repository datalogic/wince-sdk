WiFiGetGlobalSettings

Gets all global settings. The get->structSize field must be set to desired size (e.g. sizeof(GlobalSettings)) otherwise the function will return SDCERR_INVALID_PARAMETER.

 

SDCERR WiFiGetGlobalSettings(GlobalSettings *get);

 

Parameters

GlobalSettings *get

Struct large enough to contain all global values.

 

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on general failure

SDCERR_INVALID_PARAMETER if buffer is not large enough or get is NULL

Fills structSize with desired size if get is non NULL and not large enough

 

Remarks

Supported devices: Skorpio X4, Falcon X4
