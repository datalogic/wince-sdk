Compiling and Linking

 

The JoyaCradleAPI is made up of direct Win32 style function calls, which are exported by a DLL. To use this JoyaCradleAPI .DLL in your application, do the following:

 

  1. In your Embedded Visual C++ project, add the associated JoyaCradleAPI .LIB to the list of import libraries.

  2. In each source file which makes calls to the JoyaCradle API , include the associated header file(s), JoyaCradle.h
