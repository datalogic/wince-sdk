KeybdCustomSetList

 

Sets a new key custom key association list to apply to the keypad.

 

BOOL cdecl KeybdCustomSetList (CUSTOM_KEY* list, DWORD length);

 

Parameters

CUSTOM_KEY* list

This parameter points to a CUSTOM_KEYarray that is used to set the keypad’s custom key associations. This function does nothing and returns FALSE if this parameter is null.

DWORD length

This parameter is the length of the array (in number of elements) pointed to by the “list” parameter. This function does nothing and returns FALSE if this parameter is 0.

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

When this function is successfully executed, any previously-defined custom key association list is cleared and the entire custom key list subsequently consists only of the list just set.

For more information about CUSTOM_KEY see section Data Structures.
