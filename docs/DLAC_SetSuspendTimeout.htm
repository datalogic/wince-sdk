DLAC_SetSuspendTimeout

It sets the automatic AC Suspend Timeout value.

 

BOOL DLAC_SetSuspendTimeout(int param)

 

Parameters

param

integer value representing the AC Suspend Timeout to be set in second.  

Return Value

TRUE if success, FALSE in case of failure.

 

 

Requirements

Header:

declared in DL_DeviceApi.h, include DL_DeviceApi.h.

Library:

use DL_API.lib.

 

Remarks

This function writes the parameter in the Windows registry:

KEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Power\Timeouts\ACSuspendTimeout.
