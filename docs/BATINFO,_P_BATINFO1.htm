BATINFO, *P_BATINFO

 

The structure holds the Smart Battery information.  

Please refer to Defined Values for information on BATTERY_SERIAL_NUMBER_SIZE and BATTERY_MFG_NAME_SIZE.

 

Members:

DWORD capacity;

The estimated capacity of the main battery in mAh.

WORD year;

The year the main battery was manufactured.

WORD week;

The week the battery was manufactured.

WCHAR serialNumber[BATTERY_SERIAL_NUMBER_SIZE];

Buffer that will receive the battery serial number. The string returned will be null terminated.

WCHAR mfgName[BATTERY_MFG_NAME_SIZE];

Buffer that will receive the null terminated string specifying the battery manufacturer.
