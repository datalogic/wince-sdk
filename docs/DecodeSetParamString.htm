DecodeSetParamString

 

BOOL cdecl DecodeSetParamString (DWORD index, WCHAR* data, DWORD length);

 

Parameters:

DWORD index

The value of the item to be set.

 

WCHAR* data

The pointer to the Unicode string buffer that contains the data characters to set.

 

DWORD length

The length of the string passed in (divide by two to get the number of WCHAR characters).

 

Returns:

Returns TRUE if the data was copied else returns FALSE.

Remarks:

The acceptable index arguments are string of Unicode characters as indicated in General Decoding Definitions.

The index argument can assume one of the following values:

LABEL_PREFIX

LABEL_SUFFIX

LABEL_SEPARATOR.
