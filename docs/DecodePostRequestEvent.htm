DecodePostRequestEvent

 

This posts a request for data from a decoding device. The function will return immediately, and signal the specified event when the request is fulfilled.

 

DWORD cdecl DecodePostRequestEvent( HANDLE hFile, DWORD reqType, HANDLE hEvent );

 

Parameters:

HANDLE hFile

DWORD reqType

Type of request being posted. It is either a number of samples to collect or an OR'ing of any of the values listed in section

Request Types.  This ORâ€™ing is made up of the Number of data samples to collect (1-255) with whether it is a recurring request.  For a recurring event the number of samples is ignored.

HANDLE hEvent

Handle to an event to signal when the request is fulfilled.

 

Returns:

On success, a request ID is returned. On error, -1 is returned.

 

Remarks:

This should be used in a subsequent call to DecodeReadString and/or DecodeCancelRequest.
