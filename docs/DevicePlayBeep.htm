DevicePlayBeep

 

 

BOOL DevicePlayBeep(BEEP_TONE tone, BEEP_TIME duration, BOOL async);

 

 

Parameters

BEEP_TONE tone

Specifies what tone is to be played (see section Beep Tone).

BEEP_TIME duration

Specifies the duration the beep is to be played (see section Beep Time).

BOOL async

Specifies whether this function should return immediately or wait for the sound to finish playing.  If TRUE, then this function returns immediately, else this function will return when the sound has completed playback.

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
