Class Param

 

public class Param

extends java.lang.Object

 

Param allows developers to set and get values associated with various functions/symbologies of the terminal. This acts as a wrapper around DecodeAPI.getParam and DecodeAPI.setParam. The methods defined in this class are static and do not require the declaration of a new object.

 

For further information see R44-2362A (4410/4420 Product Reference Guide)
