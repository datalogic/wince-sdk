Good Read Interval

It is used in combination with the good read duration parameter (0x22) to control the duty cycle (blinking) of good read notifications. The good read duration parameter controls how long a notification occurs while the good read interval controls how much of a delay there should be between notifications.  The interval only effects notifications if the user has selected more than one good read notification to occur during a good read.

002E

#define GOOD_READ_INTERVAL

0: 20 milliseconds (default)

1: 70 milliseconds

2: 130 milliseconds

3: 180 milliseconds

4: 360 milliseconds
