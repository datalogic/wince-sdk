DLBATTERY_GetSuspendTimeout

 

It gets the automatic Battery Suspend Timeout value.

 

int DLBATTERY_GetSuspendTimeout()

 

 

Parameters

param

none  

 

 

Return Value

An integer representing the Battery Suspend Timeout value in second.

 

Requirements

Header:

declared in DL_DeviceApi.h, include DL_DeviceApi.h.

Library:

use DL_API.lib.

 

 

Remarks

This function reads the Windows registry:

HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Power\Timeouts\ BattSuspendTimeout.
