DeviceGetTriggerType

 

This gets the type of input data acquired when a trigger is activated.

 

DWORD cdecl DeviceGetTriggerType( DWORD trigger );

 

 

Parameters

DWORD trigger

Specifies the trigger to query. Valid values are listed in section Trigger Identification Masked Values.

 

Return Value

The return value is the input data type currently set for acquisition when the indicated trigger is activated. Valid values are listed in section Trigger action assignment values.

Zero is returned if the trigger is disabled, or there is an error.  To determine if an error condition exists, call GetLastError.  If the last error is ERROR_SUCCESS, then the value returned from GetTriggerType indicates TRIGGERA return.
