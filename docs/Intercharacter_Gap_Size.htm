Intercharacter Gap Size

 

It sets the intercharacter gap size for Code39 and Codabar symbologies. Due to various barcode printing technologies, the gap can grow larger than the maximum allowed and this parameter allows increase the tolerance limit for this kind of barcode.

 

 

Syntax

Name:

IMAGER_PARAM_INTERCHAR_GAP

Set by:

Value 

Values:

IMAGER_PARAM_INTERCHAR_NORMAL,

IMAGER_PARAM_INTERCHAR_LARGE

Default:

IMAGER_PARAM_INTERCHAR_NORMAL

 

 

Requirements

Header:

declared in IMAGER_PARAMETERS.H, include SE_PARAMETERS.H.
