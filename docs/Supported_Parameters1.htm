Supported Parameters

 

Some general decoding parameters only  apply to a specific type of reader like an imager or a laser. Others may apply to all barcode scanners.

In order to understand which parameters can be applied with a specific type of scanner can be followed two different ways:

 

  1. Please refer to the Decoding Parameter section of this help file. There can be found the tables with the parameters description and the different readers that support them.

  2. Use the DecodeGetParamLimits function to determine which parameters apply to a specific reader. It follows a sample source code:

 

 

 

  /* Example:

   * we want to know whether the long range input mode, long range enable spot,

   * and imager illuminate enable are supported with a specific reader.

   */

 

   DCD_PARAM_LIMIT paramArray[3];

 

   paramArray[0].index = LONG_RANGE_INPUT_MODE;

   paramArray[1].index = LONG_RANGE_ENABLE_SPOT;

   paramArray[2].index = IMAGER_ILLUMINATE_ENABLE;

 

   // This gets the valid range of values for one or more label programming parameters.

   BOOL success = DecodeGetParamLimits( paramArray, 3 );

 

   //Any unsupported parameter index will be assigned limits of 0 to

   //0xFFFFFFFF.

   if ( paramArray[0].max == (DWORD)-1 )

   {

      // Then LONG_RANGE_INPUT_MODE is NOT supported!

   }
