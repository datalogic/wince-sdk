Starting Group Separator Transmission

It enables/disables the deletion of the first EAN128 group-separator character.

 

 

Syntax

Name:

IMAGER_PARAM_CODE_128_EAN_FIRST_GROUP_SEPARATOR

Set by:

Value

Values:

IMAGER_PARAM_DISABLE

 

IMAGER_PARAM_ENABLE

Default:

IMAGER_PARAM_DISABLE

 

 

Requirements

Header:

declared in IMAGER_PARAMETERS.H, include SE_PARAMETERS.H.
