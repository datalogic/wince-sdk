DecodeGetParam

 

This gets the value of a single label programming parameter.

 

DWORD cdecl DecodeGetParam( DWORD index );

 

Parameters:

DWORD index

The index of the label programming parameter to get. Refer to section Decoding Parameters for more information on programmable settings for Datalogic devices.

 

Returns:

On success, the value of the parameter is returned. On error, including an unsupported parameter index, -1 is returned.
