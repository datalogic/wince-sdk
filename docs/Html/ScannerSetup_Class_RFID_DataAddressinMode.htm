Data Addressing Mode

 

This parameter specifies the addressing mode of the reading and writing operations on the RFID memory. By default, the reading and writing operations start from an address in bytes specified by the RF_PARAM_DATA_BUFFER_ADDRESS. The RFID tag memory is seen as a memory of successive bytes which can be addressed through an offset. Since the tag memory is made of blocks, the block addressing mode is also possible. Therefore, the address specified by the parameter RF_PARAM_DATA_BUFFER_ADDRESS will be a block number.

 

 

Syntax

Name:

RF_PARAM_DATA_BUFFER_ADDRESSING_MODE

Set by:

Value

Values:

RF_PARAM_BYTEADDR

RF_PARAM_BLOCKADDR

Default:

RF_PARAM_BYTEADDR

 

 

Requirements

Header:

declared in RF_Parameters.h, include SE_PARAMETERS.H.
