Security Level

 

Four levels of decode security are provided for linear code types (e.g., Code 39, Interleaved 2 of 5). Higher security levels are selected for decreasing levels of bar code quality.

As security levels increase, the scannerâ€™s aggressiveness decreases:

Level 1

The following code types must be successfully read twice before being decoded:

Level 2

All code types must be successfully read twice before being decoded.

 

Level 3

Code types other than the following must be successfully read twice before being decoded.

The following codes must be read three times:

Level 4

All code types must be successfully read three times before being decoded.

 

 

Syntax

Name:

BBCR_PARAM_LINEAR_CODE_TYPE_SECURITY_LEVELS

Set by:

Value

Values:

from 1 to 4

Default:

1 (Serial One Dimensional Barcode Engine) 2 (Serial Barcode Raster Engine)
