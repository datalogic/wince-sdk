EAN/UPC Supplemental Redundancy

 

If the Autodiscriminate option is enabled, this option adjusts the number of times + 1 a symbol without supplementals is decoded before transmission. Possible values are 0 and 1.

 

 

NOTE

If both this parameter and the Redundancy parameter share a value different from 0, only the one set to the highest value will be taken into account for decoding a code without add on.

 

 

Syntax

Name:

BCD_PARAM_EAN_UPC_SUPPLEMENTAL_REDUNDANCY

Set by:

Value

Values:

0      1

Default:

0

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
