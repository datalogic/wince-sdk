Label Type

 

This parameter allows selecting a label type.

 

 

Syntax

Name:

RF_PARAM_LABEL_TYPE

Set by:

Value

Values:

RF_PARAM_ISO15693 - for Iso-15693 Compliant

RF_PARAM_ISO15693_ICODE - for I-CODE SLI

RF_PARAM_ISO15693_TAGIT -for Texas Tag-It I

RF_PARAM_INFINEON - for Infineon

RF_PARAM_ISO14443A - for Iso-14443A Compliant

RF_PARAM_MIFARE_ULTRALIGHT -for Philips Mifare Ultralight Iso-14443A

RF_PARAM_MIFARE_1K - for Philips Mifare Standard Iso-14443A

RF_PARAM_MIFARE_4K - for Philips Mifare 4k Iso-14443A

RF_PARAM_PHILIPS_ICODE – for Philips I.CODE 1 tags (*)

RF_PARAM_ISO15693_FUJITSU – for Iso-15693 Fujitsu tag (*)

RF_PARAM_BARCODE - to select barcode reading only on RFID + Laser Engine

Default:

RF_PARAM_ISO15693_ICODE

 

(*) available only with R3 revision. To understand which RFID revision is onboard, call DLScannerSetup::getRfidRevision.

Requirements

Header:

declared in RF_Parameters.h, include SE_PARAMETERS.H.
