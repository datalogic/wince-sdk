Developing GAPI on Datalogic Mobile Computers

 

The gx.h and gx.lib files are available for developers who need to use GAPI 1.2 on Datalogic mobile computers.

 

 

Troubleshooting

 

Problem

Suggestion

Some applications occupy the entire display area hiding the task bar.

If the task bar is not in Auto-hide state, it can disturb the game. It is suggested to set Auto-hide in Start=>Settings=>Taskbar and StartMenu window.

Some applications install gx.dll in their installation directory or overwrite the file in Windows directory.

The installed gx.ddll file must be renamed or deleted in order to use the Datalogic gx.dll implementation. To avoid an hardware reset, it is better to maintain a copy of “\windows\gx.dll” before installing the application.
