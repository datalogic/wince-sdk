DLGSM API

 

 

For Windows CE, the Datalogic GSM (DLGSM) Library provides a basic C API for SMS message sending and for user authentication procedure by means of PIN and PUK codes.

 

For Windows Mobile, refer to the standard Library showed in the Microsoft® Windows Mobile 6 Documentation (Phone Features - CellCore).
