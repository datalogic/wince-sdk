DecodePostRequestMsg

 

This posts a request for data from a decoding device. The function will return immediately, and post a message to the specified window when the request is fulfilled.

 

DWORD cdecl DecodePostRequestMsg(HANDLE hFile, DWORD reqType, HWND hWnd, UINT uMsg);

 

Parameters:

HANDLE hFile

Handle to the decoding device to post to. The CreateFile function returns this handle.

DWORD reqType

Type of request being posted. It is either a number of samples to collect or an OR'ing of any of the values listed in section Request Types.  This ORâ€™ing is made up of the Number of data samples to collect (1-255) with whether it is a recurring request.  For a recurring event the number of samples is ignored.

HWND hWnd

Handle to a window whose window procedure is to receive a message when the request is fulfilled.

UINT uMsg

Specifies the message to be posted when the request is fulfilled. This should be a value of WM_USER (defined in winuser.h) or higher.

 

Returns:

On success, a request ID is returned. On error, -1 is returned.

 

Remarks:

This should be used in a subsequent call to DecodeReadString and/or DecodeCancelRequest.

 

When posted to the specified window, the lParam and wParam value of the message are set as follows:

 

Value

Description

lParam

The request ID of the posted request to which the input data belongs

wParam

The size, in bytes, of the input data fulfilling the request. For string data, this is the number of WCHARs and includes the terminating null character
