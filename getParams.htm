getParams

 

public static boolean getParams(Datalogic.SDK.DCD_PARAM[] paramArray)

 

Returns the values of one or more label programming parameters.  

 

Parameters:

paramArray - An array of DCD_PARAM objects indicating the parameters to get. Each array element's index member is initialized with the index of a parameter to get, and the value member is filled in upon return with the value of the parameter.

 

Returns:

True on success, otherwise false. If an unsupported index is included in the parameter array, its value will be set to -1 and the return value will be false.
