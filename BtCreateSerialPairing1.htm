BtCreateSerialPairing

 

Creates a serial pairing based on a string BT UUID of a device. BtCreateSerialPairing() can be used to create a serial printer connection.

 

BOOL cdecl BtCreateSerialPairing( WCHAR *address, int portIndex );

 

 

Parameters

WCHAR *address

Null terminated string version of UUID

 

int portIndex

COM port index 0 to MAX_PORT_NUM

 

 

Return Value

TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

On failure an additional error message can be found by calling GetLastError().

 

ERROR_BUSY

Driver is busy performing an inquiry.

ERROR_NO_MORE_DEVICES

Attempt to add more than MAX_DEVICES pairings.

ERROR_INVALID_PARAMETER

The target device does not support serial or does not exist.

 

Device must have bluetooth hardware.
