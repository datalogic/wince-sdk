BtGetPairingInfo

 

Gets pairing info for all pairs. Used in conjunction with BtGetPairingCount().

 

BOOL cdecl BtGetPairingInfo(BTPAIR *pairs, DWORD pairCnt);

 

 

Parameters

pairs

Object of size (size of BTPAIR times pairCnt) that contains information on all pairs.

 

pairCnt

Number of pairings in the pairs array (from 0 to 16).

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

Remarks

On failure an additional error message can be found by calling GetLastError().

 

ERROR_INSUFFICIENT_BUFFER

 More pairs exist than what can be filled in the pairs array

 

Device must have bluetooth hardware.
