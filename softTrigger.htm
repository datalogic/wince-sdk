softTrigger

 

public boolean softTrigger(int inputType,

                                      int timeout)

 

Activates the trigger of this DecodeHandle object's associated decoding device.

 

Parameters:

inputType - Specifies the input data type to begin acquiring. (one of the Data Input Types in APIEnum)

timeout - Amount of time, in milliseconds, to continue the attempt to acquire data. If data has not been obtained after this time period, then the trigger is deactivated.

 

Returns:

True is returned on success, otherwise false is returned. The method will return immediately.
