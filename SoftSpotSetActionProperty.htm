SoftSpotSetActionProperty

 

 

Sets a property of an action

 

BOOL cdecl SoftSpotSetActionProperty(DWORD actionId, SOFTSPOT_ACTION_PROPERTY_TYPE propertyType, WCHAR *value);

 

 

 

 

Parameters:

DWORD actionId                       

    ID of the action to set the property of

 

SOFTSPOT_ACTION_PROPERTY_TYPE propertyType        

    Which property type to set

 

WCHAR  *value             

    String containing the new value of the property

 

 

 

Returns:

TRUE if property was set successfully, FALSE otherwise
