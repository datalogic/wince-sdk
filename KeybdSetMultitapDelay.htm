KeybdSetMultitapDelay

 

Sets the delay setting used by the multitap feature when the unit is in alpha mode.

 

BOOL cdecl KeybdSetMultitapDelay( int delaySetting );

 

 

Parameters

int delaySetting

Holds the new multitap setting. Valid settings can be found in section Keyboard multitap delay definitions.

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure., (e.g. this function is not supported by the installed keypad).
