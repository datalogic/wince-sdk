readString

 

public int readString(Datalogic.SDK.DecodeString dcd,

                              int reqID,

                              boolean getCodeID)

 

Reads data that was used to fulfill a request previously posted to this DecodeHandle object.

 

Parameters:

dcd - empty DecodeString object.

reqID - Request identifier of the request to cancel. The DecodeHandle.postRequestWait and DecodeHandle.postRequestEvent methods return this identifier.

getCodeID - This boolean determines if the code ID will be returned within the DecodeString object. If this is set to false the code ID in the DecodeString object will be set to -1.

 

Returns:

On success, the number of characters copied into the String. Specifying a request ID that has not yet been fulfilled or if dcd is null -1 will be returned.
