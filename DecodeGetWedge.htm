DecodeGetWedge

 

This gets the current keyboard wedge setting for a particular input data type (barcode, MSR, etc.).

 

BOOL cdecl DecodeGetWedge( DWORD inputType );

 

Parameters:

DWORD inputType

Indicates the type of input data whose wedge setting is to be queried. Valid values are listed in section Input Data Types Mask.

Valid values are listed in section Input Data Types Mask and Wedge Type Mask.

 

Returns:

FALSE is returned if the wedge feature is disabled for the input data type, and TRUE is returned otherwise.

 

Remarks:

If an invalid input type is specified, FALSE is returned.
