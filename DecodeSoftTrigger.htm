DecodeSoftTrigger

 

This activates the trigger of a decoding device to begin data acquisition.

 

BOOL cdecl DecodeSoftTrigger( HANDLE hFile, DWORD inputType, DWORD timeout );

 

Parameters:

HANDLE hFile

Handle to the decoding device. The CreateFile function returns this handle.

DWORD inputType

Specifies the input data type to begin acquiring. Valid values are listed in section Input Data Types Mask.

DWORD timeout

Amount of time, in milliseconds, to continue the attempt to acquire data. If data has not been obtained after this time period, then the trigger is deactivated.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure. The function will return immediately.
