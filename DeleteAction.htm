DeleteAction

 

Syntax

 

   public static bool DeleteAction(UInt32 actionId);

 

 

Class                           : SoftSpot

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Deletes the action with the given ID, and removes it from list of actions SoftSpot can display.

 

 

 

Parameters:

UINT32 actionId                       

    ID of the action to delete

 

 

                                              

Returns:

true if action with given ID was deleted, false otherwise
