DecodeGetParamString

 

DWORD cdecl DecodeGetParamString (DWORD index, WCHAR* data, DWORD length);

 

Parameters:

DWORD index

The value of the item to be retrieved.

 

WCHAR* data

The pointer to the Unicode string buffer to copy the data characters into.

 

 DWORD length

The length of the buffer passed in to hold the data to be saved (divide by two to get the number of WCHAR characters).

Returns:

The actual number of bytes copied into the data buffer. Return 0 if there are no Unicode characters or if the DecodeSetParam function was used to set a single character.

Remarks:

The acceptable index arguments are string of Unicode characters as indicated in General Decoding Definitions.

The index argument can assume one of the following values:

LABEL_PREFIX

LABEL_SUFFIX

LABEL_SEPARATOR.
