setParams

 

public static boolean setParams(Datalogic.SDK.DCD_PARAM[] paramArray)

 

Sets the values of one or more label programming parameters.  

 

Parameters:

paramArray - An array of DCD_PARAM objects indicating the parameters to set. Each array element contains the index of a parameter to set, and its new value.

 

Returns:

True on success, otherwise false. If any unsupported index is used, or if an attempt is made to set a parameter to a value outside its valid limits, none of the parameter changes will take effect.
