BtEnable

 

Turns on or off the Bluetooth radio.

 

BOOL cdecl BtEnable(BOOL on);

 

 

Parameters

BOOL on

TRUE to turn on or FALSE to turn off

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.  On failure a system error message number can be found by calling GetLastError().

Remarks

Device must have bluetooth hardware.
