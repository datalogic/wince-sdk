KeybdGetMultitapDelay

 

Gets the delay setting used by the multitap feature when the unit is in alpha mode.

 

int cdecl KeybdGetMultitapDelay( void );

 

 

Return Value

One of the values listed in section Keyboard multitap delay definitions.

 

Remarks

This function will return KBD_MULTITAP_DELAY_ERROR if this function is not supported by the installed keypad.
