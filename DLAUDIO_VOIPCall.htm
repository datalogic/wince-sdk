DLAUDIO_VOIPCall

 

It signals the system to enable or disab le the audio configuration for a VOIP call.

 

BOOL DLAUDIO_VOIPCall(BOOL bEnable)

 

 

Parameters

bEnable

if TRUE, configuration for VOIP call is enabled.

if FALSE, configuration for VOIP call is disabled.

 

Return Value

TRUE indicates success, FALSE indicate failure.

 

Requirements

Header:

declared in DL_DeviceApi.h, include DL_DeviceApi.h.

Library:

use DL_API.lib.
