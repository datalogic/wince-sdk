Barcode Library

 

It gives the possibility to enable/disable the barcode library. When enabled, the captured image is passed to the barcode library and if the specific symbology is enabled the barcode can be decoded.

 

If disabled, the captured image is not passed to the barcode library and therefore no decoding can be performed even if the specific symbology is enabled.

 

The possible values are:

 

  • Enable (Default setting)

  • Disable

 

 

Syntax

Name:

IMAGER_PARAM_BARCODELIB_ENABLE

Set by:

Value

Values:

IMAGER_PARAM_DISABLE

IMAGER_PARAM_ENABLE

Default:

IMAGER_PARAM_ENABLE
