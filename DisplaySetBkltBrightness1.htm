DisplaySetBkltBrightness

 

Sets the backlight brightness value.

 

int cdecl DisplaySetBkltBrightness( int Brightness );

 

 

Parameters

int Brightness

One of the backlight brightness values listed in the section The backlight brightness values.

 

 

Return Value

One of the backlight brightness values listed in section The backlight brightness values. The return value is the current setting of the backlight. If successful the return value will match the parameter passed to the function.
