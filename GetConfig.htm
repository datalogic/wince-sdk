WiFiGetConfig

Retrieves the configuration with given name. WEP/PSK keys or EAP authenication data are not returned.

 

SDCERR WiFiGetConfig(char *name, SDCConfig *cfg);

 

Parameters

char *name

Name of the desired profile

 

SDCConfig *cfg

Configuration structure to return named profile settings

Return Value

SDCERR_SUCCESS on success.

SDCERR_INVALID_NAME if can't match name

SDCERR_INVALID_CONFIG if cfg isn't valid

 

Remarks

Supported devices: Skorpio X4, Falcon X4
