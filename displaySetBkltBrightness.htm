displaySetBkltBrightness

 

public static int displaySetBkltBrightness(int brightness)

 

Sets the backlight brightness value. See the backlight brightness definitions in APIEnum for valid parameters. This method is not supported on the 4410/4420 Monochrome.

 

Returns:

This method will return the brightness setting if no error occurred. See the backlight brightness definitions in APIEnum for return values.
