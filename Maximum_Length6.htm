Maximum Length

 

It defines the maximum length of the Matrix 2/5 codes. Refer to the Text Length definition for further details.

Syntax

Name:

IMAGER_PARAM_CODE_25_MATRIX_MAX_TEXT_LENGTH

Set by:

Value

Values:

From 0 to 55

Default:

0

 

 

Requirements

Header:

declared in IMAGER_PARAMETERS.H, include SE_PARAMETERS.H.
