GetCurrentStatus (Method)

 

Syntax

 

public static WiFiStatus GetCurrentStatus()

 

 

Class                           : WiFi

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

Description

Get current statistics for the WLAN card

 

Return

WiFiStatus class containing statistics

 

Throws

WiFiException on error.

 

 

Example

 

   WiFiStatus status = WiFi.GetCurrentStatus();

   Debug.WriteLine("cardState: " + status.CardState);

   Debug.WriteLine("client MAC: " + status.ClientMACAddress.ToString());

   Debug.WriteLine("client IP: " + status.ClientIPAddress.ToString());

   Debug.WriteLine("AP MAC: " + status.ApMACAddress.ToString());

   Debug.WriteLine("AP IP: " + status.ApIPAddress.ToString());

   Debug.WriteLine("channel: " + status.Channel);

   Debug.WriteLine("RSSI: " + status.Rssi);

   Debug.WriteLine("Driver ver: " + status.DriverVersion);

 

Remarks

Supported devices: Skorpio X4, Falcon X4

 

See Also

WiFi (Class), Example #43, WiFiStatus.
