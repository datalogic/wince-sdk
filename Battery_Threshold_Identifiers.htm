Name

Description

Identifier for the threshold level between the High and Low battery statuses for the Main battery

Identifier for the threshold level between the Low and Critical battery statuses for the Main battery

Identifier for the threshold level between the High and Low battery statuses for the Backup battery

BATTERY_BACKUP_TH_LOW_CRITICAL

Identifier for the threshold level between the Low and Critical battery statuses for the Backup battery
