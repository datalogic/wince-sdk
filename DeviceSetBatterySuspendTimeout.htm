DeviceSetBatterySuspendTimeout

 

This function allows to set the device timeout before automatic suspension when running on battery.

 

BOOL cdecl DeviceSetBatterySuspendTimeout(DWORD timeout);

 

 

Parameters

DWORD timeout

The value of the timeout (in seconds) or 0 for infinite.

Return Value

A value of TRUE is returned on success otherwise FALSE is returned to indicate failure

 

Remarks

After this function is called, the value shown by the Power applet in the Control Panel may not be correct if timeout parameter was set to a value not available in the drop-down list of such applet.
