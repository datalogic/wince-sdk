Check Digit Transmission - Code 39

 

It allows including the check digit when transmitting the Code 39 code.

 

 

Syntax

Name:

BCD_PARAM_CODE_39_STANDARD_CHECK_DIGIT_TRANSMISSION

Set by:

Value

Values:

BCD_PARAM_DISABLE

BCD_PARAM_ENABLE

Default:

BCD_PARAM_DISABLE

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
