BTH_RADIO_MODE

 

The BTH_RADIO_MODE enumeration contains the values to determine all the possible modes of operation of the Bluetooth radio.

 

 

Syntax

enum BTH_RADIO_MODE {

     BTH POWER OFF

     BTH CONNECTABLE

     BTH DISCOVERABLE

} ;

Enumerators

BTH_POWER_OFF

Turns off the Bluetooth radio.

 

BTH_CONNECTABLE

Turns on the Bluetooth radio and makes it connectable.

 

BHT_DISCOVERABLE

Turns on the Bluetooth radio and makes it both connectable and discoverable.

 

 

Requirements

Header:

declared in bthutil.h, include bthutil.h.
