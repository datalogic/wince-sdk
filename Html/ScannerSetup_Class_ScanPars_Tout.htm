Timeout

 

This is the maximum period of time, expressed in milliseconds, during which the scanner stays on without decoding any barcode.

 

 

Syntax

Name:

SCAN_PARAM_TIMEOUT

Set by:

Value

Values:

unsigned long: from 500 to 10000

Default:

5000

 

 

Requirements

Header:

declared in SCAN_PARAMETERS.H, include SE_PARAMETERS.H.

 

 

NOTE

This parameter has no effect on devices with Imager Data Capture (i.e. Datalogic J Seriesâ„¢).
