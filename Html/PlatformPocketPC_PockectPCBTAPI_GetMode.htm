BthGetMode

 

This method retrieves the current mode of operation of the Bluetooth radio.

 

 

Syntax

int BthGetMode(

  DWORD * pdwMode

) ;

 

 

Parameters

pdwMode

[out] Indicates the current mode of operation of the Bluetooth radio. See BTH_RADIO_MODE for possible values.

 

 

Return Value

Returns ERROR_SUCCESS on success, or an error code describing the error on failure.

 

 

Requirements

Header:

declared in bthutil.h, include bthutil.h.

Library:

use bthutil.lib
