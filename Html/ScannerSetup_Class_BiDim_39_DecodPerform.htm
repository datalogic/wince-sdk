Decode Performance

 

This option offers three levels of decode performance or aggressiveness for Code 39 symbols. Increasing the performance level reduces the amount of required bar code orientation, which is useful if you are scanning very long and/or truncated bar codes. Increased levels reduce decode security.

 

 

Syntax

Name:

BBCR_PARAM_CODE39_DECODE_PERFORMANCE

Set by:

Value

Values:

BBCR_PARAM_DISABLE

BBCR_PARAM_ENABLE

Default:

BBCR_PARAM_ENABLE

 

 

Requirements

Header:

declared in BBCR_PARAMETERS.H, include SE_PARAMETERS.H.
