BT_DEVICE, *BT_DEVICE

 

The structure holds information about all devices discovered.  See sections Enumerations and Defined Values for information on BT_MAX_NAME_SIZE and BT_ADDR_STR_SIZE.

 

Members:

WCHAR name[BT_MAX_NAME_SIZE];  Name of device

BT_ADDR addr;

 MAC address of the device
WCHAR addrStr[BT_ADDR_STR_SIZE];   

String version of the MAC address
DWORD cod;   Class of Device (see www.bluetooth.org)
