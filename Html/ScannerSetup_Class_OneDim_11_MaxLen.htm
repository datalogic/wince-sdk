Code 11 Maximum Length

 

It defines the maximum length of the Code 11 codes. Refer to the Text Length definition for further details.

 

 

Syntax

Name:

BCD_PARAM_CODE_11_MAX_TEXT_LENGTH

Set by:

Value

Values:

unsigned long : from 0 to 0xff, SEE TEXT LENGTH OPTION

Default:

63

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
