Exposure Mode Selection

 

It allows managing four EGC (Exposition Gain Control) algorithms:

 

  • Dummy Algorithm

  • Standard Algorithm

  • Spot Algorithm

  • Mean Algorithm

 

 

Dummy Algorithm

 

Fixed exposure mode. No EGC algorithm is used setting this parameter.

 

 

Standard Algorithm 

 

Entire Image mode. The Standard Algorithm is based on the analysis of the whole image. This mode works well in most of standard applications. It is the default setting.

 

 

Spot Algorithm

 

Central part of the image. The Spot Algorithm is based on the analysis of a restricted area positioned in the central zone of the image. This mode is suggested when reading small codes positioned in a dark and extensive background. 

 

 

Mean Algorithm

 

Highly reflective surfaces. This algorithm allows reading codes on highly reflective surfaces. This mode is suggested, for example, when reading codes positioned on plastic or metal surfaces. 

 

 

Syntax

Name:

IMAGER_PARAM_EGC_ALGORITHM

Set by:

Value

Values:

IMAGER_PARAM_EGC_DUMMY_ALGORITHM

IMAGER_PARAM_EGC_STANDARD_ALGORITHM

IMAGER_PARAM_EGC_SPOT_ALGORITHM

IMAGER_PARAM_EGC_MEAN_ALGORITHM

Default:

IMAGER_PARAM_EGC_STANDARD_ALGORITHM
