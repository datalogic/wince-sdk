Check Digit Transmission - Industrial 2/5

 

It allows including the check digit when transmitting the Industrial 2/5 code.

 

 

Syntax

Name:

BCD_PARAM_CODE_25_INDUSTRIAL_CHECK_DIGIT_TRANSMISSION

Set by:

Value

Values:

BCD_PARAM_DISABLE

BCD_PARAM_ENABLE

Default:

BCD_PARAM_DISABLE

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
