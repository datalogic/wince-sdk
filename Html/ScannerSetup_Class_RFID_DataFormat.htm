Data Format

 

This parameter specifies the reading buffer format of the Data. The format can be returned as it is or can be converted into hexadecimal format.

 

 

Syntax

Name:

RF_PARAM_DATA_BUFFER_FORMAT

Set by:

Value

Values:

RF_PARAM_ASCII_FORMAT

RF_PARAM_HEX_FORMAT

Default:

RF_PARAM_ASCII_FORMAT

 

 

Requirements

Header:

declared in RF_Parameters.h, include SE_PARAMETERS.H.
