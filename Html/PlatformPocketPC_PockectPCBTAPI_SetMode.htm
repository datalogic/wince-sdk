BthSetMode

 

This method sets the mode of operation of the Bluetooth radio and reflects in the Control Panel. This method is also used to persist this state across hardware insertion and reboot.

 

 

Syntax

int BthSetMode(

  DWORD * dwMode

) ;

 

 

Parameters

dwMode

[in] Indicates the mode of operation the Bluetooth radio should be set to. See BTH_RADIO_MODE for possible values.

 

 

Return Value

Returns ERROR_SUCCESS on success, or an error code describing the error on failure.

 

 

Requirements

Header:

declared in bthutil.h, include bthutil.h.

Library:

use bthutil.lib
