Enable Postal Code

 

It enables the mobile computer to read the Postal Code symbology.

 

If the parameter is IMAGER_PARAM_POSTAL_DISABLED, all postal symbologies are disabled. Otherwise IMAGER_PARAM_POSTAL sets which of the postal symbologies is enabled for decoding.

 

 

Syntax

Name:

IMAGER_PARAM_POSTAL

Set by:

Value

Values:

IMAGER_PARAM_POSTAL_DISABLED

IMAGER_PARAM_POSTAL_AUSTRALIA

IMAGER_PARAM_POSTAL_JAPAN

IMAGER_PARAM_POSTAL_PLANET

IMAGER_PARAM_POSTAL_POSTNET

IMAGER_PARAM_POSTAL_KIX

IMAGER_PARAM_POSTAL_ROYAL_MAIL

Default:

IMAGER_PARAM_POSTAL_DISABLED

 

 

  • IMAGER_PARAM_POSTAL_DISABLED:

all postal symbology decoding disabled.

  • IMAGER_PARAM_POSTAL_AUSTRALIA_POST:

Australia Post decoding enabled.

  • IMAGER_PARAM_POSTAL_JAPAN_POST:

Japan Post decoding enabled.

  • IMAGER_PARAM_POSTAL_PLANET:

Planet decoding enabled.

  • IMAGER_PARAM_POSTAL_POSTNET:

Postnet decoding enabled.

  • IMAGER_PARAM_POSTAL_KIX:

KIX Code decoding enabled.

  • IMAGER_PARAM_POSTAL_ROYAL_MAIL:

Royal Mail 4 State Customer Code (British Post Office) decoding enabled.
