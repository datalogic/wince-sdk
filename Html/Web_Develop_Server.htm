Web Server

Your Datalogic mobile computer can be used as a Web Server communicating with a Web Client to provide web applications.

 

The mobile computer web server is Microsoft® Internet Information Server, which supports both the IASPI technology and the ASP technology using JavaScript as unique scripting language (for further details refer to Microsoft® documentation).

 

The ASP and HTML pages created by the developer must be copied under the \Windows\www\wwwpub directory of the mobile computer file system.

 

Upon connection with any web browser by editing the machine address (ex. http://172.16.4.15) in the web browser address bar, it will be possible to navigate the previously created pages.

Furthermore, these technologies allow creating ASP pages including the ActiveX Controls which have been previously installed on the mobile computer. Thanks to these Controls, some of the mobile computer functions can be remotely managed.

 

In particular, the client/server architecture, where the Datalogic mobile computer works as Server while any device with a Web browser works as client, allows a remote control and configuration of the mobile computer by means of a simple Internet connection.
