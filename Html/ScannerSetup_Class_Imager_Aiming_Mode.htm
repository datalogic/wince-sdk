Aim Mode

 

It allows managing three aiming modes:

 

  • Timed Hold

  • Timed Release

  • Auto Capture

 

 

 

Timed Hold

 

If the user keeps the trigger button pressed for more than the Aim Duration time, the aim will switch off and the scanner will switch on to read bar codes.

 

 

Timed Release

 

If the user keeps the trigger button pressed for more than the Aim Duration time, the aim will work as in Timed Hold mode. If the user releases the trigger button before the Aim Duration time has expired, the aim will switch off and the scanner will switch on to read bar codes.

 

 

Auto Capture

 

When the user press the trigger, the driver performs a continuous image acquisition. For each acquired image the driver sets an event which can be listened from any application that can display the image.

When the user releases the trigger the driver acquires the last image that can be decoded if the decoding parameter is enabled. 

 

 

Syntax

Name:

IMAGER_PARAM_AIM_MODE

Set by:

Value

Values:

IMAGER_PARAM_AIM_TIMED_HOLD

IMAGER_PARAM_AIM_TIMED_RELEASE

IMAGER_PARAM_AIM_AUTOCAPTURE

Default:

IMAGER_PARAM_AIM_TIMED_HOLD
