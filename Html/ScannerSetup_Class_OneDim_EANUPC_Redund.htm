EAN/UPC Redundancy

 

This option adjusts the number of times + 1 a symbol is to be decoded before being accepted.

 

 

Syntax

Name:

BCD_PARAM_EAN_UPC_REDUNDANCY

Set by:

Value

Values:

from 0 to 7

Default:

7

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
