Not Printable Char

 

If enabled, this parameter allows the deletion of all not printable characters included in the scanned data. In this case the final barcode will include only printable characters.

 

 

Syntax

Name:

SCAN_PARAM_REMOVE_NOTPRINTABLE_CHAR

Set by:

Value

Values:

SCAN_PARAM_DISABLE

 

SCAN_PARAM_ENABLE

Default:

SCAN_PARAM_DISABLE

 

 

Requirements

Header:

declared in SCAN_PARAMETERS.H, include SE_PARAMETERS.H.
