Led Key Pad

 

This determines the keyboard backlight blinking after each good read. The blinking duration and the number of pulses are the same as those set for the LED parameters. 

 

 

Syntax

Name:

SCAN_PARAM_LED_KEYPAD

Set by:

Value

Values:

SCAN_PARAM_DISABLE

 

SCAN_PARAM_ENABLE

Default:

SCAN_PARAM_DISABLE

 

 

Requirements

Header:

declared in SCAN_PARAMETERS.H, include SE_PARAMETERS.H.
