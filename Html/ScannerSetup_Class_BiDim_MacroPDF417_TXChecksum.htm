Transmit Checksum

 

Transmit Checksum activates transmission of the checksum field. The field character tag is \923\006. For example, the field may be: \923\00663823.

 

 

Syntax

Name:

BBCR_PARAM_MACROPDF_TX_CHECKSUM

Set by:

Value

Values:

BBCR_PARAM_DISABLE

BBCR_PARAM_ENABLE

Default:

BBCR_PARAM_DISABLE

 

 

Requirements

Header:

declared in BBCR_PARAMETERS.H, include SE_PARAMETERS.H.
