Wireless

 

The DL_API library provides a basic C API for the device radio handling.

 

The table below shows the functions that are valid only for models using the WiFi5 radio (5xx-xxx-xxx models).

 

For models using the WiFi7 radio (7xx-xxx-xxx models), you can find information about the configuration applet: http://www.summitdatacom.com/SCU.htm.
