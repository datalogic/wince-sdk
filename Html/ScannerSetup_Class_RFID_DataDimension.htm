Data Dimension

 

This parameter specifies the number of bytes the RFID driver must read on the RFID memory. The bytes will be read starting from the address specified by the parameter RF_PARAM_DATA_BUFFER_ADDRESS.

 

 

Syntax

Name:

RF_PARAM_DATA_BUFFER_DIMENSION

Set by:

Value

Values:

unsigned long: from 1 to 65536

Default:

10

 

The maximum number of bytes an application can read on a RFID label depends on the RFID type features: it corresponds to the number of blocks multiplied by the number of bytes per block.

 

 

Requirements

Header:

declared in RF_Parameters.h, include SE_PARAMETERS.H.
