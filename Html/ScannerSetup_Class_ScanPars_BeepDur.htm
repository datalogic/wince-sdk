Beeper Duration

 

This is the time, expressed in milliseconds, during which a beeper will be emitted from the scanner at bar code reading. If set to 0 no beep will be emitted.

 

 

Syntax

Name:

SCAN_PARAM_BEEPER_DURATION

Set by:

Value

Values:

unsigned long: from 0 to 200

Default:

20

 

 

Requirements

Header:

declared in SCAN_PARAMETERS.H, include SE_PARAMETERS.H.

 

 

NOTE

This parameter has no effect on Datalogic J Series™, Datalogic Skorpio™ and Datalogic Memor™ 4.XX.
