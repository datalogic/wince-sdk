Check Digit Type - MSI

 

It allows enabling the check digit control and its type for the MSI codes among:

  • none

  • MOD 10

  • MOD 11, MOD 10

  • MOD 10, MOD 10

 

 

Syntax

Name:

BCD_PARAM_CODE_MSI_CHECK_DIGIT_TYPE

Set by:

Value

Values:

BCD_PARAM_CODE_MSI_CHECK_MOD_10

BCD_PARAM_CODE_MSI_CHECK_11_MOD_10

BCD_PARAM_CODE_MSI_CHECK_10_MOD_10

BCD_PARAM_CODE_MSI_CHECK_NONE

Default:

BCD_PARAM_CODE_MSI_CHECK_MOD_10

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
