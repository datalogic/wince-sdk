Code Severity Level

 

It sets the severity level to be applied to the barcode decoding. The lower this value is, the more aggressive the decoding is. While an high severity level corresponds to a more severe decoding.

 

 

Syntax

Name:

BCD_PARAM_LINEAR_CODE_SEVERITY

Set by:

Value

Values:

BCD_PARAM_MIN_SEVERITY

BCD_PARAM_LOW_SEVERITY

BCD_PARAM_MID_SEVERITY

BCD_PARAM_HIGH_SEVERITY

BCD_PARAM_MAX_SEVERITY

Default:

BCD_PARAM_MID_SEVERITY

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
