Codes Per Scan

 

This parameter sets the number of codes located on the same scan line to be read. Codes are concatenated through a separator string selected by the user. If the number of codes on the scan line is less than BCD_PARAM_CODES_PER_SCAN, no codes are decoded.

 

 

Syntax

Name:

BCD_PARAM_CODES_PER_SCAN

Set by:

Value

Values:

An integer value between BCD_PARAM_MIN_CODES_PER_SCAN and BCD_PARAM_MAX_CODES_PER_SCAN

Default:

BCD_PARAM_MIN_CODES_PER_SCAN

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
