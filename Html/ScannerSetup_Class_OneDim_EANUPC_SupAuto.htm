EAN/UPC Autodiscriminate

 

It enables the decoding of both codes with and without Add on.

 

WARNING

This parameter is valid only if Add on 2 or Add on 5 is also enabled.

 

 

Syntax

Name:

BCD_PARAM_EAN_UPC_SUPPLEMENTAL_AUTODISCRIMINATE

Set by:

Value

Values:

BCD_PARAM_DISABLE

BCD_PARAM_ENABLE

Default:

BCD_PARAM_DISABLE

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
