Beeper Type

 

This determines the beeper type in case of good read. It is possible to define a single tone, a double tone or a user-configurable tone by using the SCAN_PARAM_BEEPER_DURATION and SCAN_PARAM_BEEPER_FREQUENCY parameters.

 

 

Syntax

Name:

SCAN_PARAM_BEEPER_TYPE

Set by:

Value

Values:

SCAN_PARAM_BPR_DUAL_TONE,

 

SCAN_PARAM_BPR_MONO_TONE

Default:

SCAN_PARAM_BPR_DUAL_TONE

 

 

Requirements

Header:

declared in SCAN_PARAMETERS.H, include SE_PARAMETERS.H.

 

 

NOTE

This parameter has no effect on Datalogic J Series™, Datalogic Skorpio™ and Datalogic Memor™ 4.XX.
