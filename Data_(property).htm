Datalogic .NET SDK

Data (Property)

 

Syntax

 

   public AccData Data {get; }

 

 

Sub Class                      AccEventArgs

Class                           : Device

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Description

Returns the accelerometer data sampled by the device when the DataReady event was raised.

 

 

Example

 

void accStream_DataReady(object sender, Device.AccEventArgs e)

{

      this.AccDataX.Text = e.Data.x.ToString("f2");

      this.AccDataY.Text = e.Data.y.ToString("f2");

      this.AccDataZ.Text = e.Data.z.ToString("f2");

      this.timeStamp.Text = e.Data.timeStampMs.ToString();

}
