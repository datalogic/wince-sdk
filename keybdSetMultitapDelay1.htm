keybdSetMultitapDelay

 

public static boolean keybdSetMultitapDelay(int delaySetting)

 

Sets the delay setting used by the multitap feature when the unit is in alpha mode.

 

Parameters:

delaySetting - Specifies the new multitap setting. See the keyboard multitap timeout definitions in APIEnum.

 

Returns:

Returns true if delay was set, false otherwise. This method will return false if this method is not supported by the installed keypad.
