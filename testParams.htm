testParams

 

public static long testParams(Datalogic.SDK.DCD_PARAM[] paramArray)

 

Tests the valid range of values for one or more label programming parameters.  

 

Parameters:

paramArray - An array of DCD_PARAM objects indicating the parameters to test. Each array element contains the index and new value of a parameter to test.

 

Returns:

Zero is returned if all parameters and values are valid, otherwise non-zero is returned. If an invalid entry exists, the return value is the parameter index with the error, OR'd together with one of the Parameter Limit Indicator codes.
