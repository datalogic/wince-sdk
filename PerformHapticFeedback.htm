DeviceVibrate

Provide haptic vibrator feedback to the user.                                                    

 

BOOL cdecl DeviceVibrate(HAPTIC_ID hapticType);

 

 

Parameters:

HAPTIC_ID * hapticType  Type of haptic response to perform. Valid values are listed in section Haptic_ID

 

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure. FALSE will be returned if device does not have haptic vibrator hardware.
