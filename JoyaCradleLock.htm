JoyaCradleLock

It manages Joya cradle lock/unlock commands.

 

BOOL cdecl JoyaCradleLock(JOYA_CRADLE_LOCK_TYPE lockType)

 

Parameters

JOYA_CRADLE_LOCK_TYPE lockType

 

Cradle lock/unlock command specified in JOYA_CRADLE_LOCK_TYPE

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
