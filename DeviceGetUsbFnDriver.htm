DeviceGetUsbFnDriver

Gets the current USB function driver.

 

USBFN_CLASS cdecl DeviceGetUsbFnDriver(USBFN_STATE state);

 

Parameters

USBFN_STATE state

Are you querying for the current/active driver, or what will be loaded after the device is rebooted?

Return Value

USBFN_CLASS value of the chosen state

The value returned could be different depending on the 'state' value passed in, particularly if you issued a DeviceSetUsbFnDriver call, but haven't rebooted yet.

Remarks

Supported devices: Skorpio X4, Falcon X4
