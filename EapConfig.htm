DEPRECATED

This structure is present in the SDK, but the SDCConfig struct should be used instead to setup EAP profiles.

EAPConfig

EAP configuration struct

 

 

Members:

 

char SSID[SSID_SZ];

Access Point SSID

DWORD Broadcast;

Network broadcast setting

1 = connect to the network, even if it's not broadcasting the SSID

0 = only connect to the network if it's SSID is being broadcast

DWORD Protect;

Only applicable for PSK profiles that use a password.

When adding a profile, this field should be set to 0.

 

When editing a profile:

1 = re-use existing password

0 = use provided new password

char ImportPACFilePath[261];

Full path to the PAC file for manual PAC provisioning. Protected Access Credentials (PAC) files are credentials used to prove the device's identity to a Radius server in the EAP-FASTconnection method.

 

Example: "\Windows\test.pac"

char pacAIDinfo[256];

PAC authority identity (AID) information used to select PAC from a list of PACs on the device.

 

Used during EAP-FAST connections. All EAP-FAST authenticators are identified by an AID.

DWORD Cckm;

Cisco Centralized Key Management (CCKM) Fast Roaming.

1 = yes, using CCKM

0 = no, not using CCKM

DOT11_CIPHER_ALGORITHM CipherAlg;

Cipher algorithm (encryption type) to use for the connection

DWORD EapReserved1;

Reserved for future use

char SharedKey[1025];

shared secret key value

DOT11_AUTH_ALGORITHM AuthAlg;

auth algorithm (handshaking) type

BOOL EapReserved2;

Reserved fro future use

DWORD PmkCache;

Use Pairwise Master Key (PMK) caching?

Allows authenticating clients can skip EAP exchange while roaming between access points.

1 = yes, use PMK caching

0 = no, don't use PMK caching

EapTypes EapType;

EAP method to use

CredentialsType CredType

credentials type

CredentialsSource CredSource;

credentials source

DWORD ValidateServer;

Whether or not the server should be validated

1 = validate server certificate

0 = do not validate server certificate

char ServerCertName[256];

"Friendly Name" of the server digital certificate that should be validated. Certificate must already be loaded on the system and visible in the Certificates Control Panel under "Trusted Authorities"

char ClientCertName[256];

"Friendly Name" of the client digital certificate that should be used. Certificate must already be loaded on the system and visible in the Certificates Control Panel under "My Certificates"

DWORD InnerMethod;

inner EAP method to use

DWORD AutoPAC;

Automatic PAC provisioning (EAP-FAST setting)

0 = Use manual PAC

1 = Use automatic PAC

DWORD EncryptID;

Encrypt Identity of EAP methods?

1 = Encrypt identity of EAP methods
char UserName[256];

username of EAP method

char Password[256];

password of EAP method
