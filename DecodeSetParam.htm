DecodeSetParam

 

This sets the value of a single label programming parameter.

 

BOOL cdecl DecodeSetParam( DWORD index, DWORD value );

 

Parameters:

DWORD index

The index of the label programming parameter to set.  Refer to section Decoding Parameters for more information on programmable settings for Datalogic devices.

DWORD value

The new value of the parameter.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks:

Zero will be returned if an unsupported index is used, or if an attempt is made to set a parameter to a value outside its valid limits. If the function fails, GetLastError will return one of the error codes listed in Section Error Codes.

Parameters which do not have values (such as DEFAULTS_FACTORY) may be supplied as input.  In this case, the value parameter is ignored, and the value of all parameters are updated based on the parameter chosen.  For example, calling DecodeSetParam(DEFAULTS_FACTORY, 0) will set all parameters to factory defaults.

Refer to section Decoding Parameters for more information on programmable settings for Datalogic devices.
