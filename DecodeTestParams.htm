DecodeTestParams

 

This tests the valid range of values for one or more label programming parameters.

 

long cdecl DecodeTestParams( DCD_PARAM * paramArray, int paramCount );

 

Parameters:

DCD_PARAM * paramArray

Pointer to an array of DCD_PARAM structures (see section Data Structures) indicating the parameters to test. Each array element contains the index and new value of a parameter to test.

int paramCount

Number of elements in the array pointed to by paramArray.

 

Returns:

Zero is returned if all parameters and values are valid, otherwise non-zero is returned.

 

Remarks:

If an invalid entry exists, the return value is the parameter index with the error, OR'ed together with one of the Parameter Limit Indicator codes listed in section Parameter Limit Bits.
