KeybdLoadKeymap

 

Loads Keyboard Map from a file on the unit.

 

int cdecl KeybdLoadKeymap( WCHAR* keyMapPath );

 

Parameters

WCHAR* keyMapPath

Pointer to a null terminated string that stores the path and filename of the Keyboard Map. Default argument loads the default configuration.

 

Return Value

This function returns an error code. Error codes are defined in Keyboard Remap Error Definitions.
