getTriggerType

 

public static int getTriggerType(int trigger)

 

Returns the type of input data acquired when a trigger is activated.  

 

Parameters:

trigger - Specifies the trigger to query. (one of the Trigger Identifiers in APIEnum).

 

Returns:

The return value is the input data type currently set for acquisition when the indicated trigger is activated. (one of the Input Data Types).
