postRequestEvent

 

public int postRequestEvent(int reqType)

 

Posts a request for data from the decoding device associated with this DecodeHandle object. The method will return immediately.

 

Parameters:

reqType - Type of request being posted. It is an OR'ing of any of the Request Type values listed in APIEnum.

 

Returns:

On success, a request ID is returned. On error, APIEnum.INVALID_HANDLE is returned.
