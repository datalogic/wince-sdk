Class Summary

 

APIEnum

APIEnum contains terminal related constant values for use within the Datalogic.SDK package.

DCD_PARAM

DCD_PARAM is a data structure used to retrieve or set the value of a given parameter on the terminal.

DCD_PARAM_LIMIT

DCD_PARAM_LIMIT is a data structure used to retrieve the limits for a given parameter on the terminal.

DecodeAPI

DecodeAPI gives developers access to decoder related methods of the terminal.

Class DecodeException

DecodeException is used for decoder related exceptions.

DecodeHandle

DecodeHandle is used to access a decoding device on the terminal.

DecoderEvent

DecoderEvent class, used by DecodeHandle.

DecodeString

DecodeString is a data structure used with DecodeHandle.readString to retrieve scanned data from the decoder.

DeviceAPI

DeviceAPI gives developers access to unit and keyboard related functions of the terminal.

Param

Param allows developers to set and get values associated with various functions/symbologies of the terminal.
