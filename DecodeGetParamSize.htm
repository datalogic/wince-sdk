DecodeGetParamLength

 

DWORD cdecl DecodeGetParamLength(DWORD index);

 

Parameters:

DWORD index

The index of the label programming parameter to get the length of.

 

Returns:

On success, the string length (number of WCHARs) of the parameter's value is returned.  On error, including a parameter index that does not support string values, -1 is returned
