Compiling and linking

 

The DecodeAPI is made up of direct Win32 style function calls, which are exported by DecodeAPI.DLL. The DLL has an associated .LIB file and .H file.  To use this DLL in an application, the following should be sufficient.

  1. In your Visual Studio project, add the DecodeAPI.LIB to the list of import libraries.

  2. In each source file which makes calls to the API, include the appropriate header file, DecodeAPI.h and CommonAPI.h.
