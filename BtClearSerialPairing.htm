BtClearSerialPairing

 

Destroys a serial pairing.

 

BOOL cdecl BtClearSerialPairing( WCHAR *address );

 

 

Parameters

WCHAR *address

string version of UUID

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

On failure an additional error message can be found by calling GetLastError().

 

Return Values

Description

ERROR_INVALID_PARAMETER

The device does not support serial or does not exist.

 

Device must have bluetooth hardware.
