WiFiGetRadioEnable

Returns the radio state

 

SDCERR WiFiGetRadioEnable(BOOL *state);

 

Parameters

BOOL *state

Pointer to a BOOL. The function will set the value to TRUE if the Radio is enabled, or to FALSE if the radio is not enabled.

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on failure

 

Remarks

Supported devices: Skorpio X4, Falcon X4
