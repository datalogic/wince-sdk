DeviceEnableModem

 

This function allows to switch on/off the cradle add-on (either modem or Ethernet).

 

BOOL cdecl DeviceEnableModem(BOOL bEnable);

 

 

Parameters

BOOL bEnable

TRUE to turn on, FALSE to turn off.

 

Return Value

 A value of TRUE is returned on success otherwise FALSE is returned to indicate failure.

 

Remarks

Device must be on the cradle, otherwise function will fail.

If no add-on is installed, the function is still successful, but has no effect.

A port of the main cradle is disabled when activating the add-on, and re-enabled when the add-on is turned off again. Namely, microUSB is not available when Ethernet add-on is in use; same goes for serial port and modem add-on.
