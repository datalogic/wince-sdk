JoyaCradleLock(Method)

 

Syntax

 

   public static Boolean JoyaCradleLock(Device.JoyaCradleLockCmd lockCmd );

 

 

Class                           : Device

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Description

Send lock/unlock commands to JoyaCradle

 

 

Parameters

Device.JoyaCradleLockCmd

        public enum JoyaCradleLockCmd

        {

            LOCK = 0,

            LOCK_WITH_LED_OFF,

            UNLOCK,

            UNLOCK_WITH_LED_ON,

        }

 

 

 

Return Value

A value of true is returned on success, otherwise false is returned to indicate failure.
