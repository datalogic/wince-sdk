JoyaCradleReadConfigArea

It reads a custom byte array from Joya Cradle Persistent Memory dedicated to custom data.

 

BOOL  cdecl  JoyaCradleReadConfigArea(BYTE* area, unsigned long size, unsigned long* pReturnSize);

 

Parameters

BYTE* area

Array of byte to read.

unsigned long size

Size of the array (max value is 575)

unsigned long* pReturnSize

Returned size of the array.

where

typedef union

{

BYTE fastChargeEnabled;

BYTE data[CONFIG_AREA_SIZE];

} JOYA_CRADLE_CONFIG;

 

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
