Description

AIM Code ID Modifiers for (Micro)PDF-417.

 

 

Name

Value

Description
Normal

0

Normal symbology label; no special processing.

ECI

1

ECI protocol support.

BCI

2

 BCI protocol support.
C128_FNC1_1st

3

Code 128 emulation; implied FNC1 is first character.
C128_FNC1_2nd

4

Code 128 emulation; implied FNC1 is second character.
C128  

5

Code 128 emulation; no implied FNC1.
