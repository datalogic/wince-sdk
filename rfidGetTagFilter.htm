rfidGetTagFilter

 

public Datalogic.SDK.TagMemory rfidGetTagFilter()

 

This gets the tag types supported by a decoding device.

 

Returns:

The return value is a combination of bits, which indicate the tag types supported by the decoding device.

 

 

This gets the tag filter used for read operations.

 

Returns:

A TagMemory object containing the current filter being used for read operations. null is returned on error.
