Compiling and Linking

 

The SoftSpot API is made up of direct Win32 style function calls, which are exported by a DLL. To use this SoftSpotApi.DLL in your application do the following:  

 

1. In  your Embedded Visual C++ project, add the associated SoftSpotApi.LIB to the list of import libraries.

2. In each source file which makes calls to the SoftSpot API, include the associated header file, SoftSpotApi.h.
