SoftSpotGetPosition

 

 

Get the position, as a percentage of screen width and height of the center of the SoftSpot.

 

BOOL cdecl SoftSpotGetPosition(POINT* pt);

 

 

 

 

Parameters:

POINT* pt                     

    Pointer to a POINT structure that will be filled in with the current position of the SoftSpot on screen, as a percentage of screen width and height.

 

 

 

Returns:

TRUE if pt has been filled with the valid position of SoftSpot, FALSE otherwise
