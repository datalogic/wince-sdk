SoftSpotAddAction

 

 

Adds a new action to list of actions SoftSpot can cycle through and perform.

 

DWORD cdecl SoftSpotAddAction(SoftSpotAction* newAction);

 

 

 

 

Parameters:

SoftSpotAction* newAction         

    A pointer to a new SoftSpotAction to add.If actionType is SOFTSPOT_ACTION_SCAN, the application and arguments fields in newAction struct are not used.

    For all action types, the actionId and order fields are not used.

 

 

 

Returns:

ID of newly created action. If the add was not successful, a value of SOFTSPOT_ACTION_ID_INVALID is returned
