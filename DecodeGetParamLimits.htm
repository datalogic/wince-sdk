DecodeGetParamLimits

 

This gets the valid range of values for one or more label programming parameters.

 

BOOL cdecl DecodeGetParamLimits( DCD_PARAM_LIMIT * paramArray, int paramCount );

 

Parameters:

DCD_PARAM_LIMIT * paramArray

Pointer to an array of DCD_PARAM_LIMIT structures (see section Data Structures) indicating the parameters to get limits for. Each array element's index member is initialized with the index of a parameter of interest, and the rest of the structure filled in upon return with the minimum and maximum values that parameter can take.

int paramCount

Number of elements in the array pointed to by paramArray.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks:

Any unsupported parameter index will be assigned limits of 0 to 0xFFFFFFFF, but the return value will still be TRUE.
