DecodeSetWedge

 

This enables or disables the keyboard wedge feature for a particular type of input data.

BOOL cdecl DecodeSetWedge( DWORD inputType, BOOL enable );

 

Parameters:

DWORD inputType

Indicates the type of input data whose wedge setting is to be altered. Valid values are listed in section Input Data Types Mask.

Valid values are listed in section Input Data Types Mask and Wedge Type Mask.

BOOL enable

Specifies whether or not the wedge feature is enabled. If FALSE, the feature is disabled, otherwise it is enabled.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks:

If an invalid input type is specified, FALSE is returned.
