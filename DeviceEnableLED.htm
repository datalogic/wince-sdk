DeviceEnableLED

 

 

BOOL DeviceEnableLED(LED_ID led, BOOL enable);

 

Parameters

LED_ID led

Specifies which LED to control (see section Led Identifiers).

BOOL enable

Specified whether the LED should be turned lit or not.  If TRUE, the LED is lit, else it is not.

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
