EAPConnectionConfig

EAP connection settings common to most EAP methods

 

Members:

 

BOOL ValidateServer

Whether or not the server should be validated

1 = validate server certificate

0 = do not validate server certificate

BOOL ConnectToServer

Connect to only the servers defined in the ConnectToServerNmaes char array?

1 = yes

0 = no

char ConnectToServerNames[EAP_SERVERNAMES_SZ]

Semi-colon-separated list of servers that are valid to connect to during server certificate validation. Only checked if ConnectToServer is set to 1.

char ServerCertName[EAP_COMMON_SZ]

"Friendly Name" of the server digital certificate that should be validated. Certificate must already be loaded on the system and visible in the Certificates Control Panel under "Trusted Authorities"

char OuterIdentity[EAP_COMMON_SZ]

Outer identity string. Typically, this value should be set to "anonymous". If this array is null (that is, OuterIdentity[0] == '\0'), then an separate outer identity will not be used.
