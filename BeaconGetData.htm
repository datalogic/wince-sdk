BeaconGetData

 

This function should be called by an application after itâ€™s registered event has been signaled.  This function returns oldest beacon scanned that has not been retrieved yet by the application.

 

HANDLE cdecl BeaconGetData(HANDLE beaconHandle, BT_BEACON *beacon);

 

Parameters

HANDLE beaconHandle

Handle returned by BeaconStartScan

BT_BEACON *beacon

Pointer to a BT_BEACON structure that will be filled with beacon data.

Return Value

On success, returns TRUE, else returns FALSE.  On error call GetLastError() for more information.  If  GetLastError() returns ERROR_NO_DATA then all data has been retrieved.

Remarks

Supported Devices: Joya Touch
