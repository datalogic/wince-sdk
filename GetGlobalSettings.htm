Datalogic .NET SDK

GetGlobalSettings (Method)

 

Syntax

 

public static WiFiGlobalSettings GetGlobalSettings()

 

 

Class                           : WiFi

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

Description

Gets all global settings

 

Return

WiFiGlobalSettings object containing all current global settings values

 

Throws

WiFiException on error.

 

Example

 

            WiFiGlobalSettings globals = WiFi.GetGlobalSettings();

            Console.WriteLine("Roaming RSSI Threshold: " + globals.RoamRssiThreshold + " dBm");

            Console.WriteLine("Minimum Connection Time: " + globals.RoamMinConnTime + " s");

            Console.WriteLine("Roaming RSSI Difference: " + globals.RoamRssiDiff + " dBm");

            Console.WriteLine("Data Retry Threshold: " + globals.RoamRetryThreshold + " retries");

            Console.WriteLine("Background Scan Level: " + globals.BgScanLevel + " dBm");

            Console.WriteLine("Background Scan Interval: " + globals.BgScanInterval + " s");

            Console.WriteLine("Power Save: " + (globals.PowerSaveEnable ? "Enabled" : "Disabled"));

            Console.WriteLine("Phy Mode: " + globals.PhyMode);

            Console.WriteLine("CCX: " + (globals.CcxEnable ? "Enabled" : "Disabled"));

            Console.WriteLine("CCKM: " + (globals.CckmEnable ? "Enabled" : "Disabled"));

 

 

Remarks

Supported devices: Skorpio X4, Falcon X4
