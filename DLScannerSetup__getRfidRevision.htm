DLScannerSetup::getRfidRevision

 

It provides the RFID engine model because different tags are supported depending on the RFID revision. To get extended error information, call DLScannerSetup::getStatus.

 

 

int DLScannerSetup::getRfidRevision(void)

 

 

Parameters

None

 

 

Return Value

Returns the Rfid revision: 0 is for R1 model, 1 is for R3 model.

 

 

Requirements

Header:

declared in DLScannerSetup.h, include DL_SEDAPI.h.
