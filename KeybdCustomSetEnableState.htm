KeybdCustomSetEnableState

 

This function sets the enable state of the custom key functionality. If the functionality is enabled any existing custom key association list is applied to key presses of the device. If the functionality is disabled, key presses are not modified, even if a custom key association list exists.

 

BOOL cdecl KeybdCustomSetEnableState (BOOL state);

 

Parameters

BOOL state  

This parameter is a BOOL value that sets the custom key functionality to be enabled or disabled. A value of true enables the functionality and a value of FALSE disables the functionality.

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure .

 

Remarks

This function has no effect on the actual custom key association list of the keypad. It only enables or disables the list from applying to the keypad key codes.
