WiFiEnableConfig

Set given profile's ability to connect. Other profiles are unchanged.

 

SDCERR WiFiEnableConfig(char *name, BOOL enable);

 

Parameters

char *name

Name of the desired profile

BOOL enable

New enabled state profile

 

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on general failure

SDCERR_INVALID_NAME if can't match name

 

Remarks

Supported devices: Skorpio X4, Falcon X4
