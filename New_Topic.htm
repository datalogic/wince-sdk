NOTIS Editing

 

It enables to strip the start and the stop characters from a Codabar decoded barcode.

 

Syntax

Name:

IMAGER_PARAM_CODABAR_NOTIS

Set by:

Value

Values:

IMAGER_PARAM_DISABLE,

IMAGER_PARAM_ENABLE

Default:

IMAGER_PARAM_DISABLE

 

 

Requirements

Header:

declared in IMAGER_PARAMETERS.H, include SE_PARAMETERS.H.
