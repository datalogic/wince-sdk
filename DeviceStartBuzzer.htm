DevicBuzzerStart

This produces a buzzer sound.

 

BOOL cdecl DeviceBuzzerStart(void);

 

Parameters

None.

 

 

Return Value

A value of TRUE is returned on success otherwise FALSE is returned to indicate failure. FALSE will be returned if device does not have buzzer hardware.

 

Remarks

Buzzer sound continues until DeviceBuzzerStop is called or device reboots.
