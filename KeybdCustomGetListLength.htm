KeybdCustomGetListLength

 

Gets the length of the custom key association list array (in number of elements) that currently applies to the keypad.

 

int cdecl KeybdCustomGetListLength (void);

 

Parameters

None

 

Return Value

The return value is the length of the custom key association list array (in number of elements) that currently applies to the keypad.

 

Remarks

For more information about CUSTOM_KEY see section Data Structures.
