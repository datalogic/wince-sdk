Enable Standard 2/5

 

It enables the Standard 2/5 symbology called also Discrete 2/5.

Syntax

Name:

IMAGER_PARAM_CODE_25_DISCRETE

Set by:

Value

Values:

IMAGER_PARAM_DISABLE,

IMAGER_PARAM_ENABLE

Default:

IMAGER_PARAM_DISABLE

 

 

Requirements

Header:

declared in IMAGER_PARAMETERS.H, include SE_PARAMETERS.H.
