keybdGetMultitapDelay

 

public static int keybdGetMultitapDelay()

 

Returns a number identifying the delay setting used by the multitap feature when the unit is in alpha mode.

 

Returns:

The current multitap definition (if supported). See the keyboard multitap timeout definitions in APIEnum. This method will return APIEnum.KBD_MULTITAP_DELAY_ERROR if this method is not supported by the installed keypad.
