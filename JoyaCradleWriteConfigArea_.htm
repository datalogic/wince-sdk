JoyaCradleWriteConfigArea (Method)

 

Syntax

 

   public static Boolean JoyaCradleWriteConfigArea (Byte[] area);

 

 

Class                           : Device

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Description

It writess the config area from Joya Cradle Persistent Memory dedicated to cradle configuration data. First byte is the FastCharge enabling. Other bytes are to be specified for the future.

 

Parameters

Byte[] area

Array of 32 byte to be write.

 

 

 

 

Return Value

A value of true is returned on success, otherwise false is returned to indicate failure.
