rfidSetAntenna

 

public boolean rfidSetAntenna(int antennaBits)

 

This enables the RFID antennas of a decoding device for transmit and/or receive.

 

Parameters:

antennaBits - The new RFID antenna settings. Valid values are any combination of bits using the RFID Antenna Selection methods rfidAntennaRX and rfidAntennaTX. Antennas which are not present on the decoding device will be ignored. At least one antenna each for transmit and receive must be specified.

 

Returns:

True is returned on success, otherwise false is returned.
