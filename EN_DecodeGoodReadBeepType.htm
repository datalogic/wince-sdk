DecodeGoodReadBeepType (Enumeration)

 

Syntax

 

   [Flags]

   public enum DecodeGoodReadBeepType: uint

   {

      LabelRead = 0x00,

      TagRead = 0x01,

      TagWrite = 0x00,

      TagWriteFailed = 0x03,

   }

 

 

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Description

The different Beep types which can be played using Decode.PlayGoodReadBeep().

 

 

Name

Value

Description

LabelRead

0x00

Bar code label is read.

TagRead

0x01

RFID tag is read.

TagWrite

0x02

RFID tag write succeeded.

TagWriteFailed

0x03

RFID tag write failed.
