DecodeGetCodeID

 

This function reads the code identifier of a data sample prior to calling a DecodeRead… function. It allows the application to filter incoming samples by the code identifier.

 

CODE_ID cdecl DecodeGetCodeID( HANDLE hFile, DWORD reqID );

 

Parameters:

HANDLE hFile

Handle to the decoding device with associated image buffer. The CreateFile function returns this handle.

DWORD reqID

Request identifier of the request that has been satisfied with data to be read. The DecodePostRequest… functions return this identifier.

 

Returns:

On success, the code identifier associated with the request is returned.  On failure the value CODE_ID. CODE_ID_NONE, is returned. See Section Enumerations for details of the CODE_ID enumerated type.

 

Remarks:

This function should be called after notification of a data sample. If this function is called when no sample is pending an error is returned.
