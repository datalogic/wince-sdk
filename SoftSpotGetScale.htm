SoftSpotGetScale

 

 

Get the current size (scale) of the SoftSpot.

 

BOOL cdecl SoftSpotGetScale(float* scale);

 

 

Parameters:

float *scale       

    Pointer to a float that will be filled in with a value between SOFTSPOT_SCALE_MIN and SOFTSPOT_SCALE_MAX that indicates the current scaling factor of the SoftSpot.

    For example, 1.2 would be 20% larger than the default size (which is 1.0).

 

  Returns:                       TRUE if curent scale was able to be retreived, FALSE otherwise
