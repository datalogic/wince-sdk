GS1 DataBar Expanded Maximum Length

 

It defines the maximum length of the GS1 DataBar Expanded codes. Refer to the Text Length  definition for further details.

 

Syntax

Name:

BCD_PARAM_RSS_EXP_MAX_TEXT_LENGTH

Set by:

Value

Values:

From 0 to 74

Default:

74

 

 

Requirements

Header:

declared in BCD_PARAMETERS.H, include SE_PARAMETERS.H.
