addActionListener

 

public boolean addActionListener(Datalogic.SDK.DecodeActionListener listener,

                                                    int eventID)

 

Adds a DecodeActionListener to be called when an event associated with this listener is triggered. If this listener is used to handle decoder events, the removeActionListeners or removeActionListener method should be called before the application closes to properly shut down the listener.

 

Parameters:

listener - The DecodeActionListener that will have decoderEventOccurred called when an event fires.

eventID - The event identifier to associate this DecodeActionListener object to. Event identifiers can be created with the createEvent method.

 

Returns:

True if the listener was removed, false if the event id was invalid.
