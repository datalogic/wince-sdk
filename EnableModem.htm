EnableModem (Method)

 

Syntax

 

   public static Boolean EnableModem(Boolean Enable);

 

 

Class                           : Device

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Description

Enables or disables the communication interface module that can be attached to the device's cradle.

 

 

Parameters Table

 

Type

Parameter

Description

[in]

Boolean

Enable

Set to true to enable the module or false to disable the module.

 

 

 

 

[returns]

Boolean

 

 

True is returned on success, otherwise false is returned to indicate failure.

[on error]

FALSE
