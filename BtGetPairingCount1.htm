BtGetPairingCount

 

Gets number of pairings. Used in conjunction with BtGetPairingInfo(â€¦).

 

DWORD cdecl BtGetPairingCount(void);

 

 

Return Value

On success, the number of pairings in the driver (from 0 to MAX_DEVICES) is returned. -1 is returned on failure.  On failure a system error message number can be found by calling GetLastError().

Remarks

Device must have bluetooth hardware.
