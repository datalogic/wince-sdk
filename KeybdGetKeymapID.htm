KeybdGetKeymapID

 

Retreives the ID of the currently loaded Keyboard Keymap.

 

BOOL cdecl KeybdGetKeymapID( WCHAR * uniqueID, int bufferLen);

 

 

Parameters

WCHAR * uniqueID

Points to a buffer where a string containing the keymap ID can be returned.  The buffer should be at least UNIQUEID_SIZE (see section Minimum Input Array Lengths) characters in length.

int bufferLen

Length, in characters, of the buffer allocated for uniqueID.

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
