SoftSpotSetPosition

 

 

Set the position, as a percentage of screen width and height of the center of the SoftSpot.

 

BOOL cdecl SoftSpotSetPosition(POINT pt);

 

 

 

 

 

Parameters:

POINT pt                     

    A point (x and y as a percentage of screen width and height to set the SoftSpot to)

 

 

 

Returns:

TTRUE if SoftSpot has been moved to position pt, FALSE otherwise
