cancelRequest

 

public boolean cancelRequest(int reqID)

 

Cancels a request previously posted to this DecodeHandle object.

 

Parameters:

reqID - Request identifier of the request to cancel. The DecodeHandle.postRequestWait and DecodeHandle.postRequestEvent methods return this identifier.

 

Returns:

True is returned on success, otherwise false is returned. Specifying a request ID that has not been posted will return false.
