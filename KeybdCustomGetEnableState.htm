KeybdCustomGetEnableState

 

Gets the enable state of the custom key functionality.

 

BOOL cdecl KeybdCustomGetEnableState (void);

 

Parameters
None

 

Return Value

A value of TRUE is returned if the custom key functionality is enabled. The value of FALSE is returned if the custom key functionality is disabled or if the function call had an error and did not succeed.

 

Remarks

This function has no effect on the actual custom key association list of the keypad. It only enables or disables the list from applying to the keypad key codes.

For more information about CUSTOM_KEY see section Data Structures.
