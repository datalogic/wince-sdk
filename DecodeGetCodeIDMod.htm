DecodeGetCodeIDMod

 

This function reads the AIM code ID modifier of a data sample prior to calling DecodeReadString.  It should only be called after notification of a data sample.

 

DWORD cdecl DecodeGetCodeIDMod(HANDLE hFile, DWORD reqID)

 

Parameters:

hFile

Handle to the decoding device to read data from.  The CreateFile function returns this handle.

reqID

Request identifier of the request that has been satisfied with data to be read.  The DecodePostRequest functions return this identifier.

 

Returns:

On success, the AIM code ID modifier associated with the request is returned.  See the AIM Symbology Identifier specification for details.  On error, -1 is returned.

 

Remarks:

The AIM code ID is specified as a string in the format "]Cm", where "C" is a character representing the symbology, and "m" is a modifier. The value of the modifier is the number returned by this function. It has a meaning which is specific to the label symbology. The label symbology can be determined by calling DecodeGetCodeID.
