Light Level

 

It allows managing four light levels:

 

  • Maximum light

  • Medium light

  • Low light

  • Light off

 

 

Syntax

Name:

IMAGER_PARAM_LIGHT_LEVEL

Set by:

Value

Values:

IMAGER_PARAM_LIGHT_MAX

IMAGER_PARAM_LIGHT_MEDIUM

IMAGER_PARAM_LIGHT_LOW

IMAGER_PARAM_LIGHT_OFF

Default:

IMAGER_PARAM_LIGHT_MAX
