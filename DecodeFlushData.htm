DecodeFlushData

 

This removes all input data, which is not currently fulfilling a request, from the decoding driver's queue.

 

BOOL cdecl DecodeFlushData( HANDLE hFile );

 

Parameters:

HANDLE hFile

Handle to the decoding device to flush. The CreateFile function returns this handle.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
