AccelerometerStartEvent

This posts a request for data from the Accelerometer GSensor. The function will return immediately, and signal the specified event when the request is fulfilled.

 

BOOL cdecl AccelerometerStartEvent( HANDLE hAccDevice, DWORD dwPollingInterval, HANDLE hEvent);

 

 

Parameters

HANDLE hAccDevice

Handle to the Accelerometer GSensor returned by AccelerometerOpen()

DWORD dwPollingInterval

The accelerometer polling interval in units of milliseconds. This value will be rounded to the nearest 10ms increment by the driver

HANDLE hEvent

Handle to an event to signal when the request is fulfilled

 

 

Return Value

On success, TRUE is returned. On error, FALSE is returned.

 

Remarks

 

An application should either use AccelerometerStartMsg or AccelerometerStartEvent.
