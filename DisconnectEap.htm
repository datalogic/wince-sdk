DEPRECATED

This function is present in the SDK, but it has been deprecated.

There is currently no generic function to disconnect a profile. The closest options would be to either:

  • delete the active profile using the WiFiDeleteConfig() function

  • temporarily disable the radio using the WiFiRadioDisable() function

 

WiFiConnectEap

Disconnects from the EAP profile

 

SDCERR WiFiDisconnectEap(void);

 

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on failure

 

Remarks

Supported devices: Skorpio X4, Falcon X4
