DEPRECATED

This function is present in the SDK, but the WiFiActivateConfig() should be used instead to connect all profiles.

 

WiFiConnectEap

Connects to the EAP profile

 

SDCERR WiFiConnectEap(void);

 

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on failure

 

Remarks

Supported devices: Skorpio X4, Falcon X4
