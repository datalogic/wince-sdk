DeviceGetMetrics

 

This function retrieves data that is stored on the device to keep track of performance.

 

BOOL cdecl DeviceGetMetrics ( DEVICE_METRICS * metricsArray, int metricsCount );

 

 

Parameters

DEVICE_METRICS * metricsArray

Pointer to an array of DEVICE_METRICS structures (see section DeviceGetMetrics) indicating the performance of all measured metrics.

int metricsCount

Number of elements in the array pointed to by metricsArray.

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
