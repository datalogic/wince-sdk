WiFiActivateConfig

Sets given profile to connect and all others not to connect

 

SDCERR WiFiActivateConfig(char *name);

 

Parameters

char *name

Name of the desired profile

 

Return Value

SDCERR_SUCCESS on success.

SDCERR_FAIL on general failure

SDCERR_INVALID_NAME if can't match name

 

Remarks

Supported devices: Skorpio X4, Falcon X4
