DeviceGetFlashFileRoot

 

This function retrieves the Flash Drive file system name. For example, for the F4410, "FlashFX Disk" will be returned. The purpose of this function is to aid in writing applications that function across all the Datalogic device models.

 

BOOL cdecl DeviceGetFlashFileRoot( WCHAR * rootPath, int bufferLen );

 

 

Parameters

WCHAR * rootPath

Pointer to a buffer that will receive the Flash Drive file system name. The buffer should be at least FLASH_ROOT_SIZE (see section Minimum Input Array Lengths) characters in length.  The string returned will be null terminated.

int bufferLen

Number of characters allocated for the rootPath buffer.

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

If successful, then the buffer pointed to by rootPath contains the root path, if failed then rootPath contains an empty string (the first element is the null termination character).
