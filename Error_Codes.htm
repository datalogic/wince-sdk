Error Codes

 

When many of the functions return an error code, calling GetLastError may return additional error information.   These values should be valid HRESULTs, made up as follows:

 

#define FACILITY_DCD 0xA0

#define MAKE_DCD_ERROR(code) (MAKE_HRESULT(1, FACILITY_DCD, (code))|(1UL<<29))
