getParamLimits

 

public static boolean getParamLimits(Datalogic.SDK.DCD_PARAM_LIMIT[] paramLimitArray)

 

Returns the valid range of values for one or more label programming parameters.  

 

Parameters:

paramLimitArray - An array of DCD_PARAM_LIMIT objects indicating the parameters to get limits for. Each array element's index member is initialized with the index of a parameter of interest, and the rest of the structure filled in upon return with the minimum and maximum values that parameter can take.

 

Returns:

True on success, otherwise false. Any unsupported parameter index will be assigned limits of 0 to 0xFFFFFFFF and the return value will be false.
