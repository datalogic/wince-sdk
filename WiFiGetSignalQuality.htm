WiFiGetSignalQuality

 

Get the signal quality for WiFi.

 

BOOL cdecl WiFiGetSignalQuality( DWORD * signalQuality );

 

Parameters

DWORD * signalQuality

The signal quality of the WiFi, returned as a value in the range of 0 to 31.

 

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
