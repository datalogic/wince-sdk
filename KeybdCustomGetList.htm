KeybdCustomGetList

 

Gets custom key association list that currently applies to the keypad.

 

int cdecl KeybdCustomGetList (CUSTOM_KEY* list, DWORD length);

 

Parameters

CUSTOM_KEY* list  

This parameter points to a CUSTOM_KEY array that receives the custom key association list that applies to the keypad. If this parameter is null, the function exits with no action and returns zero.

DWORD length

This parameter holds the length of the array (in number of elements) pointed to by the “list” parameter. If there are more elements in the custom key association list than specified by this parameter, the function fills the entire caller’s array with a subset of the custom keys. If there are fewer elements in the custom key association list than specified by this parameter, the function will copy only the elements that exist and will leave the rest of the caller’s array unmodified.

 

Return Value

The returned signed integer value specifies the number of array elements that were copied into the caller’s array. If this value is equal to -1, it indicates that an error occurred.

Remarks

It may be necessary to call the “KeybdCustomGetListLength” API function before calling this function so that the length of the custom key association list is known.

For more information about CUSTOM_KEY see section Data Structures.
