AccelerometerOpen

Opens a handle to the Accelerometer GSensor.

 

HANDLE AccelerometerOpen (LPCWSTR deviceName);

 

 

Parameters

LPCWSTR deviceName

Pointer to string containing the name of the Accelerometer driver (currently, the value is L“DGS1:”)

 

Return Value

Returns a unique handle to the Accelerometer GSensor. INVALID_HANDLE_VALUE indicates failure.

 

Remarks

Use the AccelerometerClose() to close this handle. The driver or SDK will implement a message queue for each open connection from an application. Only up to 9 simultaneous open connections to the Accelerometer GSensor driver are allowed.
