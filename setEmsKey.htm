DLMIFARE::setEmsKey

 

It modifies the reader configuration in order to support the EMS HMS Mifare tags. It does not execute any operation to the RF module.

Starting from the first call of this function, DLMIFARE::getKeys returns FALSE.

 

 

bool DLMIFARE::setEmsKey( unsigned long nSector )

 

 

Parameters

nSector

number of the sector where the keys are changed.

 

 

Return Value

true if success, false in case of failure.

 

 

Requirements

Header:

declared in DLMifare.h, include DL_SEDAPI.h.

Library:

use SEDAPI.lib.
