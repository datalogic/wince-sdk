public static boolean getWedge(int inputType)

 

Returns the current keyboard wedge setting for a particular input data type (barcode, MSR, etc.).  

 

Parameters:

inputType - This gets the current keyboard wedge setting for a particular input data type. (one of the Input Data Types in APIEnum).

 

Returns:

False is returned if the wedge feature is disabled for the input data type, and true is returned otherwise. If an invalid input type is specified, false is returned.
