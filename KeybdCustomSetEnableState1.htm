KKeybdCustomSetEnableState  (Method)

 

Syntax

 

   public static Boolean KeybdCustomSetEnableState(Boolean Enable);  

 

 

Class                           : Device

Namespace                : Datalogic.API

Assembly                   : Datalogic.API.DLL

 

 

Description

Sets the enable state of the custom key functionality. Does not affect the custom key association list that applies to the device.

 

Parameters Table

 

Type

Parameter

Description

[in]

 Boolean Enable Set to true to enable and false to disable.

 

 

 

 

[returns]

Boolean

 

On success, TRUE is returned.

[on error]

FALSE

 

FALSE is returned to indicate failure
