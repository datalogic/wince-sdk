Inverse 1D

 

It allows to decode regular 1D barcodes only or inverse 1D barcodes only or both regular and inverse.

 

 

Syntax

Name:

IMAGER_PARAM_INVERSE_1D

Set by:

Value 

Values:

IMAGER_PARAM_INVERSE_REG_ONLY,

IMAGER_PARAM_INVERSE_INV_ONLY,

IMAGER_PARAM_INVERSE_AUTODET

Default:

IMAGER_PARAM_INVERSE_REG_ONLY

 

 

Requirements

Header:

declared in IMAGER_PARAMETERS.H, include SE_PARAMETERS.H.
