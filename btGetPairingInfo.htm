btGetPairingInfo

 

public static byte[] btGetPairingInfo(int iCount)

 

Get pairing info for all pairs. Used in conjunction with btGetPairingCount().

 

Returns:

byte[] containing the pairing info. If the pairing info is not set, null is returned. Pairing info is a set of 616 byte chunks. For each chunk: 64bit UUID at byte 0 COD starts at byte 8 String address starts at byte 12 Integer device name starts at byte 40 Boolean Authentication starts at byte 580 Boolean Enctryption starts at byte 584 String Port starts at byte 588 Integer Portindex starts at byte 602 Integer Direction starts at byte 606
