DeviceSetUsbFnDriver

Sets the current USB function driver. Takes effect after device reboot.

 

BOOL cdecl DeviceSetUsbFnDriver(USBFN_CLASS newDriver);

 

Parameters

USBFN_CLASS newDriver

The new driver to load

Return Value

TRUE on success.

FALSE on failure

 

Remarks

Supported devices: Skorpio X4, Falcon X4

 

See also: Control Panel > PC Connection
