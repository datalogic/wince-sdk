DecodePostRequestWait

 

This posts a request for data from a decoding device. The function will block the calling thread and not return until the request is fulfilled.

 

DWORD cdecl DecodePostRequestWait( HANDLE hFile, DWORD reqType );

 

Parameters:

HANDLE hFile

Handle to the decoding device to post to. The CreateFile function returns this handle.

DWORD reqType

Type of request being posted. It is an OR'ing of any of the values listed in section Request Types.  This ORâ€™ing is made up of the Number of data samples to collect (1-255) with whether it is a recurring or monitoring request.

 

Returns:

On success, a request ID is returned. On error, -1 is returned.

 

Remarks:

This should be used in a subsequent call to DecodeReadString.
