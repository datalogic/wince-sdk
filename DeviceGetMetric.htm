DeviceGetMetric

 

This function retrieves system metrics. This function has been reserved for future use.  

 

DWORD cdecl DeviceGetMetric (DWORD index);

 

Parameters

DWORD index

The index of the desired metric. This function has been reserved for future use.  No valid indexes are currently defined.

 

Return Value

A DWORD value corresponding to the index is returned on success, otherwise a ‘-1’ is returned.

 

Remarks

If an index is included in the in the metrics array which is not supported on the device then its value will be set to -1. Since value is a DWORD, cast as an INT32 to check for the ‘-1’.
