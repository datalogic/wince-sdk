DecodeEnumDevices

 

This gets the capabilities of all decoding devices.

 

BOOL cdecl DecodeEnumDevices( DWORD * deviceCaps );

 

Parameters:

DWORD * deviceCaps

Pointer to an array of nine DWORDs. DecodeEnumDevices will store the capabilities of each decoding device in one of the DWORDs, beginning with DCD1: in the first element, and ending with DCD9: in the last element. The values stored are combinations of bits which indicate the capabilities of the given decoding device. Bits are OR'ed together from the list in the section Device Capabilities.

 

Returns:

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.
