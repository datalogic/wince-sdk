SoftSpotStart

 

 

Activates SoftSpot

 

 

BOOL cdecl SoftSpotStart(BOOL hidden);

 

 

 

 

 

Parameters:

BOOL hidden                    

    Boolean value that controls whether the SoftSpot is hidden or displayed on start

 

 

 

Returns:

TRUE if SoftSpot was started or is already running.   FALSE if SoftSpot was not started
