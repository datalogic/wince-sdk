DeviceSetTriggerType

 

This sets the type of input data acquired when a trigger is activated.

 

BOOL cdecl DeviceSetTriggerType( DWORD trigger, DWORD inputType );

 

 

Parameters

DWORD trigger

Specifies the trigger to set. Valid values are listed in Section  Trigger Identification Masked Values.

DWORD inputType

Specifies the input data type to begin acquiring when the trigger is activated. Valid values are listed in section Trigger action assignment values. Zero may also be used to disable the trigger.

Return Value

A value of TRUE is returned on success, otherwise FALSE is returned to indicate failure.

 

Remarks

Specifying an invalid trigger or input type will return FALSE.
